## 🎯 О проекте

Это комплексное учебное хранилище заметок, разработанное на основе методологии **Zettelkasten** с интеграцией **Anki** для интервальных повторений. Проект фокусируется на глубоком изучении ключевых дисциплин и подготовке к поступлению в PhD-программу.

**🧭 Цель:** подготовка к PhD (STEM)-2027  
**🧱 Метод:** Zettelkasten  
**🔁 Система повторений:** Anki  
**🗂️ Платформа:** Obsidian  
**⚙️ Статус:** активно разрабатывается  

---

## 📚 Основные области изучения

| Область | Описание |
|---------|----------|
| **Core (ML)** | Машинное обучение и фундаментальные концепции |
| **Linear Algebra** | Линейная алгебра — математическая база ML |
| **Mathematical Analysis** | Анализ и математический аппарат |
| **Probability** | Теория вероятностей и статистика |
| **Deep Learning (Y-HandBook)** | Глубокое обучение по специализированному справочнику |
| **Research Papers** | Анализ и конспектирование статей с arXiv |

## 💡 Ключевые особенности

✨ **Zettelkasten методология**  
Атомарные заметки с явными двунаправленными ссылками для построения знаний.

🔗 **Карты содержимого (MOC)**  
Каждая тема имеет обзорную заметку, которая связывает все относящиеся к ней концепты.

🎯 **Интеграция с Anki**  
Автоматическое преобразование заметок в карточки для интервальных повторений.

📊 **Систематизированное обучение**  
Структурированный путь от базовых концепций к продвинутым темам.
