## Суть

**Разность множеств** — это операция, которая создаёт новое множество, содержащее элементы первого множества, не принадлежащие второму. Обозначается символом $\setminus$ или $-$.

Для множеств $A$ и $B$ их разность $A \setminus B$ (читается: «$A$ минус $B$») состоит из тех элементов $A$, которые не содержатся в $B$. Это «вычитание» множества $B$ из множества $A$.

---

## Детали

### Формальное определение

Пусть $A$ и $B$ — подмножества множества $M$. Тогда:

$$A \setminus B := {x \in M \mid (x \in A) \land (x \notin B)}$$

**Читается**: «$A$ минус $B$ — это множество элементов из $M$, которые принадлежат $A$ и не принадлежат $B$».

### Эквивалентное определение через принадлежность

$$x \in (A \setminus B) \iff (x \in A) \land (x \notin B)$$

Элемент принадлежит разности тогда и только тогда, когда он принадлежит первому множеству, но не принадлежит второму.

### Альтернативные обозначения

- $A \setminus B$ — основное обозначение;
- $A - B$ — альтернативное обозначение;
- $A \sim B$ — редко используемое обозначение.

### Геометрическая интерпретация

На диаграммах Эйлера–Венна разность $A \setminus B$ изображается как **область $A$, не покрытая $B$**.

```
    ┌─────────┐
    │////A////│
    │///┌─────┼─────┐
    │///│     │     │
    └───┼─────┘     │
        │     B     │
        └───────────┘
```

### Свойства разности

**1. Некоммутативность**: $$A \setminus B \neq B \setminus A \quad \text{(в общем случае)}$$

**2. Неассоциативность**: $$(A \setminus B) \setminus C \neq A \setminus (B \setminus C) \quad \text{(в общем случае)}$$

**3. Разность с пустым множеством**: $$A \setminus \emptyset = A$$ $$\emptyset \setminus A = \emptyset$$

**4. Разность с самим собой**: $$A \setminus A = \emptyset$$

**5. Связь с включением**: $$(A \setminus B) \subset A$$ Разность всегда является подмножеством уменьшаемого.

**6. Связь с пересечением**: $$A \setminus B = A \setminus (A \cap B)$$

**7. Связь с объединением**: $$A \setminus B = A \cap C_M B$$ (если рассматривать в объемлющем множестве $M$)

### Разность и включение

**Если $B \subset A$**: $$A \setminus B \cup B = A$$ $$A \setminus B \cap B = \emptyset$$

**Если $A \cap B = \emptyset$**: $$A \setminus B = A$$

**Если $A \subset B$**: $$A \setminus B = \emptyset$$

### Связь с дополнением

Дополнение — это частный случай разности: $$C_M A = M \setminus A$$

где $M$ — объемлющее множество.

### Формулы с разностью

$$A \setminus (B \cup C) = (A \setminus B) \cap (A \setminus C)$$ $$A \setminus (B \cap C) = (A \setminus B) \cup (A \setminus C)$$

$$(A \cup B) \setminus C = (A \setminus C) \cup (B \setminus C)$$ $$(A \cap B) \setminus C = (A \setminus C) \cap (B \setminus C)$$

## Примеры

### Конечные множества

**Числовые примеры**: $${1, 2, 3, 4, 5} \setminus {3, 4, 5, 6} = {1, 2}$$ $${1, 2, 3} \setminus {4, 5, 6} = {1, 2, 3}$$ $${1, 2, 3} \setminus {1, 2, 3} = \emptyset$$

**С пустым множеством**: $${a, b, c} \setminus \emptyset = {a, b, c}$$ $$\emptyset \setminus {a, b, c} = \emptyset$$

**Некоммутативность**: $${1, 2, 3} \setminus {2, 3, 4} = {1}$$ $${2, 3, 4} \setminus {1, 2, 3} = {4}$$

### Числовые множества

**Стандартные разности**: $$\mathbb{Z} \setminus \mathbb{N} = {\ldots, -2, -1, 0}$$ (неположительные целые)

$$\mathbb{R} \setminus \mathbb{Q} = \text{иррациональные числа}$$

$$\mathbb{N} \setminus {1} = {2, 3, 4, \ldots}$$

$$\mathbb{R} \setminus {0} = \mathbb{R}^*$$ (ненулевые действительные)

### Интервалы

**Разность интервалов**: $$[0, 5] \setminus [2, 3] = [0, 2) \cup (3, 5]$$ $$[0, 2] \setminus [1, 3] = [0, 1)$$ $$(0, 1) \setminus {0.5} = (0, 0.5) \cup (0.5, 1)$$

**Луч минус точка**: $$[0, +\infty) \setminus {0} = (0, +\infty)$$

### Геометрические множества

**Фигуры на плоскости**:

- Круг $\setminus$ Вписанный квадрат = кольцевая область с вырезанным квадратом;
- Плоскость $\setminus$ Прямая = две полуплоскости (несвязное множество);
- Квадрат $\setminus$ Его граница = внутренность квадрата.

### Применение в определениях

**Проколотая окрестность**: $$\mathring{U}_\delta(x_0) = U_\delta(x_0) \setminus {x_0}$$

**Внутренность интервала**: $$\text{Int}([a, b]) = [a, b] \setminus {a, b} = (a, b)$$

### Последовательные разности

$$A \setminus B \setminus C = (A \setminus B) \setminus C$$

Например: $${1, 2, 3, 4} \setminus {2, 3} \setminus {1} = {1, 4} \setminus {1} = {4}$$

### Симметрическая разность

Родственная операция — **симметрическая разность** (хотя в тексте не упоминается): $$A \triangle B = (A \setminus B) \cup (B \setminus A) = (A \cup B) \setminus (A \cap B)$$
