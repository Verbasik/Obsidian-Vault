## Суть

**Диагональ декартова квадрата** — это подмножество $\Delta \subset X^2 = X \times X$, состоящее из всех упорядоченных пар с равными компонентами:

$$\Delta = {(x_1, x_2) \in X^2 \mid x_1 = x_2}$$

Геометрически (для $X = \mathbb{R}$) это прямая $y = x$ на плоскости — главная диагональ координатной плоскости.

---

## Детали

### Формальное определение

$$\Delta = {(x_1, x_2) \in X \times X \mid x_1 = x_2}$$

или эквивалентно:

$$\Delta = {(x, x) \mid x \in X}$$

**Читается**: «Диагональ — это множество всех пар вида $(x, x)$, где $x$ пробегает всё множество $X$».

### Альтернативные обозначения

- $\Delta$ — стандартное обозначение (греческая буква дельта);
- $\Delta_X$ — если нужно указать множество явно;
- $\text{diag}(X)$ — иногда используется в литературе;
- $\text{id}_X$ — в контексте отношений (отношение равенства).

### Диагональ как отношение

Диагональ представляет **отношение равенства** на множестве $X$:

$$(x_1, x_2) \in \Delta \iff x_1 = x_2$$

Это **отношение эквивалентности**:

- Рефлексивно: $(x, x) \in \Delta$ для всех $x$;
- Симметрично: если $(x, y) \in \Delta$, то $(y, x) \in \Delta$;
- Транзитивно: если $(x, y), (y, z) \in \Delta$, то $(x, z) \in \Delta$.

### Свойства диагонали

**1. Мощность**:

Для конечного множества: $$|\Delta| = |X|$$

Существует биекция $X \to \Delta$, заданная как $x \mapsto (x, x)$.

**2. Включение**: $$\Delta \subset X \times X$$

Диагональ — **собственное подмножество** декартова квадрата (если $|X| \geq 2$).

**3. Связь с проекциями**:

Для $(x_1, x_2) \in \Delta$: $$\text{pr}_1((x_1, x_2)) = \text{pr}_2((x_1, x_2))$$

То есть обе проекции совпадают.

**4. Единственность**:

Диагональ — единственное подмножество $X \times X$, где обе компоненты всегда равны.

### Дополнение диагонали

**Недиагональ** (вне диагонали): $$X^2 \setminus \Delta = {(x_1, x_2) \in X^2 \mid x_1 \neq x_2}$$

Множество всех пар с **различными** компонентами.

Для конечного множества: $$|X^2 \setminus \Delta| = |X|^2 - |X| = |X|(|X| - 1)$$

### Геометрическая интерпретация

**На числовой плоскости** $\mathbb{R}^2$: $$\Delta = {(x, y) \in \mathbb{R}^2 \mid x = y}$$

Это **прямая** $y = x$, проходящая через начало координат под углом $45°$.

**На дискретной плоскости** ${1, 2, 3}^2$: $$\Delta = {(1, 1), (2, 2), (3, 3)}$$

Геометрически: три точки на «главной диагонали» таблицы $3 \times 3$.

### Обобщения

**Диагональ в $X^n$**: $$\Delta_n = {(x_1, \ldots, x_n) \in X^n \mid x_1 = x_2 = \cdots = x_n}$$

Множество $n$-ок с равными компонентами.

**Частичная диагональ**: Множество, где равны только некоторые компоненты.

Например, в $X^3$: $${(x_1, x_2, x_3) \mid x_1 = x_2}$$

## Примеры

### Конечные множества

**Пример 1**: $$X = {a, b}$$ $$X^2 = {(a,a), (a,b), (b,a), (b,b)}$$ $$\Delta = {(a, a), (b, b)}$$

**Пример 2**: $$X = {1, 2, 3}$$ $$\Delta = {(1, 1), (2, 2), (3, 3)}$$ $$|\Delta| = 3 = |X|$$

**Таблица для ${1, 2, 3}^2$**:

```
      1   2   3
  ┌───┬───┬───┐
1 │ ✓ │   │   │
  ├───┼───┼───┤
2 │   │ ✓ │   │
  ├───┼───┼───┤
3 │   │   │ ✓ │
  └───┴───┴───┘
```

Отмеченные клетки — диагональ.

### Числовые множества

**Натуральные числа**: $$\Delta_\mathbb{N} = {(n, n) \mid n \in \mathbb{N}} = {(1,1), (2,2), (3,3), \ldots}$$

**Действительные числа**: $$\Delta_\mathbb{R} = {(x, x) \mid x \in \mathbb{R}}$$

Геометрически: прямая $y = x$ на плоскости.

**Комплексные числа**: $$\Delta_\mathbb{C} = {(z, z) \mid z \in \mathbb{C}}$$

### Геометрические примеры

**Отрезок** $X = [0, 1]$: $$\Delta = {(t, t) \mid t \in [0, 1]}$$

Геометрически: диагональ единичного квадрата $[0, 1]^2$.

**Окружность** $S^1$: $$\Delta_{S^1} = {(p, p) \mid p \in S^1}$$

Геометрически: «диагональ» тора $S^1 \times S^1$.

### Дополнение диагонали

**Для ${1, 2, 3}$**: $${1, 2, 3}^2 \setminus \Delta = {(1,2), (1,3), (2,1), (2,3), (3,1), (3,2)}$$

Все пары с различными компонентами.

$$|{1,2,3}^2 \setminus \Delta| = 9 - 3 = 6 = 3 \cdot 2$$

### Применение в отношениях

**Отношение «не равно»**: $$R_{\neq} = X^2 \setminus \Delta = {(x, y) \mid x \neq y}$$

**Отношение «равно»**: $$R_{=} = \Delta = {(x, y) \mid x = y}$$

### Диагональ в высших размерностях

**В $\mathbb{R}^3$**: $$\Delta = {(x, x, x) \mid x \in \mathbb{R}}$$

Геометрически: прямая в трёхмерном пространстве, проходящая через начало координат с направляющим вектором $(1, 1, 1)$.

**Параметризация**: $$(x, x, x) = x(1, 1, 1), \quad x \in \mathbb{R}$$
