## 2.2. Отношение включения (стр. 28–29)

### [[Кванторы в математике]]

**Связи:**

- [[Элемент множества]] — используются для формализации высказываний о принадлежности
- [[Равенство множеств]] — формализация через ∀x
- [[Отношение включения множеств]] — определение через ∀x
- [[Объединение множеств]] — формальное определение через кванторы
- [[Пересечение множеств]] — формальное определение через кванторы

### [[Равенство множеств]]

**Связи:**

- [[Множество]] — равенство определено для множеств
- [[Элемент множества]] — множества равны, если состоят из одних элементов
- [[Отношение включения множеств]] — A = B ⇔ (A ⊂ B) ∧ (B ⊂ A)
- [[Кванторы в математике]] — формальное определение через кванторы

### [[Отношение включения множеств]]

**Связи:**

- [[Множество]] — отношение между множествами
- [[Элемент множества]] — определяется через принадлежность элементов
- [[Равенство множеств]] — связь через взаимное включение
- [[Пустое множество]] — содержится в любом множестве
- [[Дополнение множества]] — связь через (A ⊂ B) ⇔ (C_M B ⊂ C_M A)
- [[Объединение множеств]] — свойства включения для объединения
- [[Пересечение множеств]] — свойства включения для пересечения

### [[Пустое множество]]

**Связи:**

- [[Множество]] — специальный вид множества без элементов
- [[Элемент множества]] — не содержит ни одного элемента
- [[Отношение включения множеств]] — подмножество любого множества
- [[Пересечение множеств]] — пересечение непересекающихся множеств
- [[Декартово произведение множеств]] — произведение с пустым множеством даёт пустое множество

## 2.3. Простейшие операции над множествами (стр. 29–33)

### Теоретико-множественные операции

#### [[Объединение множеств]]

**Связи:**

- [[Множество]] — операция над множествами
- [[Элемент множества]] — определение через принадлежность: x ∈ (A ∪ B) ⇔ (x ∈ A) ∨ (x ∈ B)
- [[Кванторы в математике]] — формализация определения
- [[Пересечение множеств]] — двойственная операция
- [[Дополнение множества]] — связь через правила де Моргана
- [[Правила де моргана для множеств]] — C_M(A ∪ B) = C_M A ∩ C_M B
- [[Свойства операций над множествами]] — ассоциативность, коммутативность, дистрибутивность

#### [[Пересечение множеств]]

**Связи:**

- [[Множество]] — операция над множествами
- [[Элемент множества]] — определение через принадлежность: x ∈ (A ∩ B) ⇔ (x ∈ A) ∧ (x ∈ B)
- [[Кванторы в математике]] — формализация определения
- [[Объединение множеств]] — двойственная операция
- [[Дополнение множества]] — связь через правила де Моргана
- [[Правила де моргана для множеств]] — C_M(A ∩ B) = C_M A ∪ C_M B
- [[Пустое множество]] — пересечение непересекающихся множеств
- [[Свойства операций над множествами]] — ассоциативность, коммутативность, дистрибутивность

#### [[Разность множеств]]

**Связи:**

- [[Множество]] — операция над множествами
- [[Элемент множества]] — определение: x ∈ (A \ B) ⇔ (x ∈ A) ∧ (x ∉ B)
- [[Дополнение множества]] — частный случай разности: C_M A = M \ A
- [[Объединение множеств]] — связь через свойства операций
- [[Пересечение множеств]] — связь через свойства операций

#### [[Дополнение множества]]

**Связи:**

- [[Множество]] — унарная операция на множестве
- [[Разность множеств]] — C_M A = M \ A
- [[Элемент множества]] — определение через непринадлежность
- [[Объединение множеств]] — связь через правила де Моргана
- [[Пересечение множеств]] — связь через правила де Моргана
- [[Правила де моргана для множеств]] — фундаментальные свойства дополнения
- [[Отношение включения множеств]] — (A ⊂ B) ⇔ (C_M B ⊂ C_M A)

#### [[Правила де моргана для множеств]]

**Связи:**

- [[Объединение множеств]] — первое правило связывает объединение с пересечением через дополнение
- [[Пересечение множеств]] — второе правило связывает пересечение с объединением через дополнение
- [[Дополнение множества]] — операция, связывающая объединение и пересечение
- [[Кванторы в математике]] — аналогия с логическими операциями ¬, ∧, ∨
- [[Свойства операций над множествами]] — демонстрация двойственности операций
