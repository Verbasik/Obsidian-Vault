## Суть

**Дополнение множества** — это множество всех элементов объемлющего множества $M$, которые не принадлежат данному множеству $A$. Обозначается $C_M A$ или просто $CA$ (когда понятно, в каком множестве берётся дополнение).

Дополнение является частным случаем разности: $C_M A = M \setminus A$. Это аналог логической операции «НЕ» (отрицания) для множеств.

---

## Детали

### Формальное определение

Пусть $A \subset M$. Тогда дополнение $A$ в $M$:

$$C_M A := M \setminus A = {x \in M \mid x \notin A}$$

**Читается**: «Дополнение $A$ в $M$ — это множество элементов из $M$, которые не принадлежат $A$».

### Обозначения

- $C_M A$ — полное обозначение (дополнение в множестве $M$);
- $CA$ или $\overline{A}$ — краткие обозначения (когда объемлющее множество ясно из контекста);
- $A^c$ — альтернативное обозначение;
- $A'$ — иногда используется в литературе.

### Геометрическая интерпретация

На диаграммах Эйлера–Венна дополнение $C_M A$ изображается как **вся область $M$, кроме $A$**.

```
╔═══════════════════════════╗
║///////////////////////////║
║////////┌───────┐//////////║
║////////│   A   │//////////║
║////////└───────┘//////////║
║///////////////////////////║
╚═══════════════════════════╝
         M
```

### Свойства дополнения

**1. Инволютивность** (двойное отрицание): $$C_M(C_M A) = A$$

**2. Дополнение пустого множества**: $$C_M \emptyset = M$$

**3. Дополнение всего множества**: $$C_M M = \emptyset$$

**4. Связь с включением** (контрапозиция): $$(A \subset B) \iff (C_M B \subset C_M A)$$ Включение «переворачивается» при взятии дополнения.

**5. Объединение с дополнением**: $$A \cup C_M A = M$$

**6. Пересечение с дополнением**: $$A \cap C_M A = \emptyset$$

Последние два свойства показывают, что $A$ и $C_M A$ образуют **разбиение** множества $M$.

### Правила де Моргана

**Первое правило**: $$C_M(A \cup B) = C_M A \cap C_M B$$ «Дополнение объединения равно пересечению дополнений».

**Второе правило**: $$C_M(A \cap B) = C_M A \cup C_M B$$ «Дополнение пересечения равно объединению дополнений».

**Обобщение на семейства**: $$C_M\left(\bigcup_{i \in I} A_i\right) = \bigcap_{i \in I} C_M A_i$$

$$C_M\left(\bigcap_{i \in I} A_i\right) = \bigcup_{i \in I} C_M A_i$$

### Дополнение и другие операции

**Дополнение разности**: $$C_M(A \setminus B) = C_M A \cup B$$

**Разность через дополнение**: $$A \setminus B = A \cap C_M B$$

**Дополнение симметрической разности**: $$C_M(A \triangle B) = (A \cap B) \cup (C_M A \cap C_M B)$$

### Зависимость от объемлющего множества

Дополнение **зависит** от выбора объемлющего множества:

Пусть $A = {1, 2}$:

- В $M_1 = {1, 2, 3}$: $C_{M_1} A = {3}$;
- В $M_2 = {1, 2, 3, 4, 5}$: $C_{M_2} A = {3, 4, 5}$;
- В $\mathbb{N}$: $C_\mathbb{N} A = {3, 4, 5, 6, \ldots}$

Поэтому важно всегда указывать, в каком множестве берётся дополнение.

## Примеры

### Конечные множества

**В конечном универсуме**: Пусть $M = {1, 2, 3, 4, 5}$: $$C_M {1, 2} = {3, 4, 5}$$ $$C_M {2, 4} = {1, 3, 5}$$ $$C_M \emptyset = {1, 2, 3, 4, 5} = M$$

**Двойное дополнение**: $$C_M(C_M {1, 2}) = C_M {3, 4, 5} = {1, 2}$$

### Числовые множества

**В множестве целых чисел**: $$C_\mathbb{Z} \mathbb{N} = {\ldots, -2, -1, 0}$$ (неположительные целые)

**В множестве действительных чисел**: $$C_\mathbb{R} \mathbb{Q} = \text{иррациональные числа}$$ $$C_\mathbb{R} [0, 1] = (-\infty, 0) \cup (1, +\infty)$$ $$C_\mathbb{R} {0} = \mathbb{R} \setminus {0} = \mathbb{R}^*$$

### Интервалы

**Дополнение интервалов на прямой**: $$C_\mathbb{R} [a, b] = (-\infty, a) \cup (b, +\infty)$$ $$C_\mathbb{R} (a, b) = (-\infty, a] \cup [b, +\infty)$$ $$C_\mathbb{R} [a, +\infty) = (-\infty, a)$$

### Правила де Моргана в действии

**Пример 1**: Пусть $M = {1, 2, 3, 4, 5}$, $A = {1, 2}$, $B = {2, 3}$.

$$A \cup B = {1, 2, 3}$$ $$C_M(A \cup B) = {4, 5}$$

$$C_M A = {3, 4, 5}$$ $$C_M B = {1, 4, 5}$$ $$C_M A \cap C_M B = {4, 5}$$

**Пример 2** (доказан в источнике): См. стр. 30-31 учебника для полного доказательства правил де Моргана.

### Геометрические примеры

**На плоскости**:

- $C_{\text{Плоскость}} \text{Круг} =$ внешность круга;
- $C_{\text{Квадрат}} \text{Диагональ} =$ квадрат без диагонали.

**В пространстве**:

- $C_{\mathbb{R}^3} \text{Шар} =$ внешность шара

### Применение в математическом анализе

**Дополнение открытых/замкнутых множеств**:

- Дополнение открытого множества — замкнутое;
- Дополнение замкнутого множества — открытое.

**Предельные точки**: точка $x$ — предельная для $A$, если любая её окрестность пересекается с $A \setminus {x}$.
