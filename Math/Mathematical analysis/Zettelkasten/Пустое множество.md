## Суть

**Пустое множество** — это множество, не содержащее ни одного элемента. Обозначается символом $\emptyset$ или ${}$.

Пустое множество уникально (существует только одно пустое множество) и является подмножеством любого множества. Это фундаментальный объект теории множеств, играющий роль «нуля» в алгебре множеств.

---

## Детали

### Определение

**Через свойство, которым не обладает ни один элемент**: $$\emptyset = {x \in M \mid x \neq x}$$

Поскольку условие $x \neq x$ ложно для любого $x$, это множество не содержит элементов.

**Через квантор**: $$\forall x , (x \notin \emptyset)$$ «Для любого объекта $x$ выполнено, что $x$ не принадлежит пустому множеству».

### Обозначения

- $\emptyset$ — основное обозначение (от норвежского слова «пустой»);
- ${}$ — альтернативное обозначение (пустые фигурные скобки);
- $\varnothing$ — иногда используется вариант с перечёркиванием.

### Уникальность пустого множества

**Теорема**: существует единственное пустое множество.

**Доказательство**: предположим, существуют два пустых множества $\emptyset_1$ и $\emptyset_2$.

- Оба не содержат элементов;
- Следовательно, для любого $x$: $(x \in \emptyset_1) \Leftrightarrow (x \in \emptyset_2)$ (обе части ложны);
- По принципу экстенсиональности: $\emptyset_1 = \emptyset_2$.

### Свойства пустого множества

**1. Универсальное подмножество**: $$\emptyset \subset A$$ для любого множества $A$.

**Объяснение**: утверждение $\forall x (x \in \emptyset \Rightarrow x \in A)$ истинно, поскольку посылка импликации всегда ложна (в $\emptyset$ нет элементов).

**2. Пустое множество — единственное подмножество самого себя**: $$\emptyset \subset \emptyset$$ $$\text{Если } A \subset \emptyset, \text{ то } A = \emptyset$$

**3. Нейтральный элемент объединения**: $$A \cup \emptyset = A$$

**4. Аннулирующий элемент пересечения**: $$A \cap \emptyset = \emptyset$$

**5. Разность с пустым множеством**: $$A \setminus \emptyset = A$$ $$\emptyset \setminus A = \emptyset$$

**6. Декартово произведение с пустым множеством**: $$A \times \emptyset = \emptyset \times A = \emptyset$$

### Мощность пустого множества

Число элементов пустого множества: $$|\emptyset| = 0$$

Число подмножеств пустого множества: $$|\mathcal{P}(\emptyset)| = 2^0 = 1$$

Единственное подмножество $\emptyset$ — это само $\emptyset$.

### Отличие от множества, содержащего пустое множество

**Важно различать**:

- $\emptyset$ — пустое множество (0 элементов)
- ${\emptyset}$ — множество, содержащее пустое множество (1 элемент)

$$\emptyset \neq {\emptyset}$$

Аналогия:

- $\emptyset$ подобно числу $0$
- ${\emptyset}$ подобно множеству ${0}$

### Пустое множество в аксиоматике

В аксиоматической теории множеств (ZF) существование пустого множества постулируется **аксиомой пустого множества**:

$$\exists A , \forall x , (x \notin A)$$

«Существует множество, которому не принадлежит ни один элемент».

## Примеры

### Задание через свойство

**Решения несовместного уравнения**: $${x \in \mathbb{R} \mid x^2 = -1} = \emptyset$$ (нет действительных решений)

$${x \in \mathbb{N} \mid x < 0} = \emptyset$$ (нет отрицательных натуральных чисел)

**Пересечение непересекающихся множеств**: $${1, 2, 3} \cap {4, 5, 6} = \emptyset$$

### Операции с пустым множеством

**Объединение**: $${1, 2, 3} \cup \emptyset = {1, 2, 3}$$ $$\emptyset \cup \emptyset = \emptyset$$

**Пересечение**: $${1, 2, 3} \cap \emptyset = \emptyset$$

**Разность**: $${1, 2, 3} \setminus \emptyset = {1, 2, 3}$$ $$\emptyset \setminus {1, 2, 3} = \emptyset$$

**Дополнение**: $$C_M \emptyset = M$$ (дополнение пустого множества — всё множество $M$)

### Включение пустого множества

$$\emptyset \subset \mathbb{N}$$ $$\emptyset \subset \mathbb{R}$$ $$\emptyset \subset {\text{любое множество}}$$

Это всегда истинно, независимо от множества.

### Степень пустого множества

$$\mathcal{P}(\emptyset) = {\emptyset}$$

Множество всех подмножеств пустого множества содержит только само пустое множество.

### Парадоксальные свойства

**Все элементы $\emptyset$ обладают любым свойством**:

Утверждение «все элементы пустого множества — простые числа» **истинно** (вакуумная истина), поскольку нет элементов, которые опровергли бы это утверждение.

Формально: $$\forall x \in \emptyset : P(x)$$ истинно для любого свойства $P$, так как $x \in \emptyset$ всегда ложно.
