## Суть

**Пересечение множеств** — это операция, которая создаёт новое множество, содержащее только те элементы, которые принадлежат **одновременно** всем исходным множествам. Обозначается символом $\cap$.

Для множеств $A$ и $B$ их пересечение $A \cap B$ состоит из тех и только тех элементов, которые содержатся и в $A$, и в $B$. Это аналог логической операции «И» (конъюнкции) для множеств.

---

## Детали

### Формальное определение

Пусть $A$ и $B$ — подмножества множества $M$. Тогда:

$$A \cap B := {x \in M \mid (x \in A) \land (x \in B)}$$

**Читается**: «$A$ пересечение $B$ — это множество элементов из $M$, которые принадлежат $A$ и принадлежат $B$».

### Эквивалентное определение через принадлежность

$$x \in (A \cap B) \iff (x \in A) \land (x \in B)$$

Элемент принадлежит пересечению тогда и только тогда, когда он принадлежит обоим множествам.

### Геометрическая интерпретация

На диаграммах Эйлера–Венна пересечение $A \cap B$ изображается как **область пересечения** двух множеств.

```
    ┌─────────┐
    │    A    │
    │   ┌─────┼─────┐
    │   │/////│     │
    └───┼─────┘     │
        │     B     │
        └───────────┘
```

### Свойства пересечения

**1. Коммутативность**: $$A \cap B = B \cap A$$

**2. Ассоциативность**: $$A \cap (B \cap C) = (A \cap B) \cap C =: A \cap B \cap C$$

**3. Идемпотентность**: $$A \cap A = A$$

**4. Аннулирующий элемент** (пустое множество): $$A \cap \emptyset = \emptyset$$

**5. Нейтральный элемент** (объемлющее множество): $$A \cap M = A \quad \text{(если } A \subset M\text{)}$$

**6. Монотонность**: $$(A \subset B) \Rightarrow (A \cap C \subset B \cap C)$$

**7. Дистрибутивность относительно объединения**: $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

### Связь с отношением включения

$$(A \cap B) \subset A$$ $$(A \cap B) \subset B$$

Пересечение является подмножеством каждого из множеств.

**Наибольшее общее подмножество**: $A \cap B$ — это наибольшее множество, содержащееся и в $A$, и в $B$.

Формально: если $C \subset A$ и $C \subset B$, то $C \subset A \cap B$.

### Связь с дополнением (правило де Моргана)

$$C_M(A \cap B) = C_M A \cup C_M B$$

«Дополнение пересечения равно объединению дополнений».

### Непересекающиеся (дизъюнктные) множества

Множества $A$ и $B$ называются **непересекающимися** или **дизъюнктными**, если: $$A \cap B = \emptyset$$

Примеры:

- ${1, 2, 3} \cap {4, 5, 6} = \emptyset$;
- Чётные числа $\cap$ Нечётные числа $= \emptyset$.

### Обобщение на семейство множеств

Пересечение произвольного семейства множеств ${A_i}_{i \in I}$:

$$\bigcap_{i \in I} A_i = {x \mid \forall i \in I : x \in A_i}$$

«Элемент принадлежит пересечению, если он принадлежит **всем** множествам семейства».

**Примеры обозначений**: $$\bigcap_{i=1}^{n} A_i = A_1 \cap A_2 \cap \ldots \cap A_n$$ $$\bigcap_{n \in \mathbb{N}} A_n$$

## Примеры

### Конечные множества

**Числовые примеры**: $${1, 2, 3} \cap {2, 3, 4} = {2, 3}$$ $${1, 2, 3} \cap {4, 5, 6} = \emptyset$$ $${a, b, c} \cap {b, c, d} = {b, c}$$

**С пустым множеством**: $${1, 2, 3} \cap \emptyset = \emptyset$$

**Идемпотентность**: $${1, 2, 3} \cap {1, 2, 3} = {1, 2, 3}$$

### Числовые множества

**Стандартные множества**: $$\mathbb{Z} \cap \mathbb{Q} = \mathbb{Z}$$ (целые числа — часть рациональных)

$$\mathbb{Q} \cap (\mathbb{R} \setminus \mathbb{Q}) = \emptyset$$ (рациональные и иррациональные не пересекаются)

$$\mathbb{R}_+ \cap \mathbb{Z} = \mathbb{N}$$ (положительные действительные ∩ целые = натуральные)

### Интервалы

**Пересечение интервалов**: $$[0, 2] \cap [1, 3] = [1, 2]$$ $$[0, 1] \cap [2, 3] = \emptyset$$ $$(0, 1) \cap [0, 1] = (0, 1)$$

**Вложенные интервалы**: $$[0, 10] \cap [2, 5] = [2, 5]$$

### Геометрические множества

**Фигуры на плоскости**:

- Круг $\cap$ Квадрат = область пересечения фигур;
- Две касающиеся окружности: пересечение = одна точка;
- Две непересекающиеся прямые на плоскости: пересечение = $\emptyset$.

### Пересечение семейств

**Конечное семейство**: $$\bigcap_{i=1}^{3} {i, i+1, i+2} = {1, 2, 3} \cap {2, 3, 4} \cap {3, 4, 5} = {3}$$

**Бесконечное семейство (вложенные отрезки)**: $$\bigcap_{n=1}^{\infty} \left[0, \frac{1}{n}\right] = {0}$$

**Бесконечное семейство (вложенные интервалы)**: $$\bigcap_{n=1}^{\infty} \left(0, \frac{1}{n}\right) = \emptyset$$

### Применение свойств

**Упрощение выражений**: $$(A \cap B) \cap A = A \cap B$$ (по идемпотентности и ассоциативности)

$$A \cap (A \cup B) = A$$ (закон поглощения)

**Дистрибутивность**: $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

Например: $${1, 2} \cap ({2, 3} \cup {1, 4}) = {1, 2} \cap {1, 2, 3, 4} = {1, 2}$$ $$({1, 2} \cap {2, 3}) \cup ({1, 2} \cap {1, 4}) = {2} \cup {1} = {1, 2}$$
