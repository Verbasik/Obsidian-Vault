## Суть

**Свойства операций над множествами** — это набор алгебраических законов, которым подчиняются теоретико-множественные операции (объединение, пересечение, дополнение). Эти свойства образуют структуру **булевой алгебры** и включают коммутативность, ассоциативность, дистрибутивность, законы де Моргана и другие тождества.

Знание этих свойств позволяет упрощать выражения с множествами и доказывать теоретико-множественные тождества.

## Детали

### Основные алгебраические свойства

Пусть $A$, $B$, $C$ — подмножества множества $M$.

#### 1. Коммутативность

**Объединение**: $$A \cup B = B \cup A$$

**Пересечение**: $$A \cap B = B \cap A$$

#### 2. Ассоциативность

**Объединение**: $$A \cup (B \cup C) = (A \cup B) \cup C$$

**Пересечение**: $$A \cap (B \cap C) = (A \cap B) \cap C$$

Благодаря ассоциативности можно писать $A \cup B \cup C$ и $A \cap B \cap C$ без скобок.

#### 3. Идемпотентность

**Объединение**: $$A \cup A = A$$

**Пересечение**: $$A \cap A = A$$

#### 4. Дистрибутивность

**Пересечение относительно объединения**: $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

**Объединение относительно пересечения**: $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$

Обе операции дистрибутивны относительно друг друга.

### Свойства нейтральных и поглощающих элементов

#### 5. Пустое множество

**Для объединения** (нейтральный элемент): $$A \cup \emptyset = A$$

**Для пересечения** (поглощающий элемент): $$A \cap \emptyset = \emptyset$$

#### 6. Универсальное множество $M$

**Для пересечения** (нейтральный элемент): $$A \cap M = A$$

**Для объединения** (поглощающий элемент): $$A \cup M = M$$

### Свойства дополнения

#### 7. Инволютивность (двойное отрицание)

$$C_M(C_M A) = A$$

Дополнение дополнения возвращает исходное множество.

#### 8. Дополнения крайних элементов

$$C_M \emptyset = M$$ $$C_M M = \emptyset$$

#### 9. Дополнение и само множество

$$A \cup C_M A = M$$ $$A \cap C_M A = \emptyset$$

Множество и его дополнение образуют **разбиение** $M$.

### Правила де Моргана

#### 10. Правила де Моргана

$$C_M(A \cup B) = C_M A \cap C_M B$$ $$C_M(A \cap B) = C_M A \cup C_M B$$

Демонстрируют **двойственность** операций $\cup$ и $\cap$.

### Законы поглощения

#### 11. Законы поглощения

$$A \cup (A \cap B) = A$$ $$A \cap (A \cup B) = A$$

### Свойства относительно включения

#### 12. Связь с включением

$$A \subset B \iff A \cup B = B$$ $$A \subset B \iff A \cap B = A$$

#### 13. Монотонность операций

$$(A \subset B) \Rightarrow (A \cup C \subset B \cup C)$$ $$(A \subset B) \Rightarrow (A \cap C \subset B \cap C)$$

#### 14. Контрапозиция дополнения

$$(A \subset B) \iff (C_M B \subset C_M A)$$

Включение «переворачивается» при взятии дополнения.

### Свойства разности

#### 15. Разность и дополнение

$$A \setminus B = A \cap C_M B$$

$$C_M A = M \setminus A$$

#### 16. Разность и включение

$$(A \setminus B) \subset A$$

$$A \setminus B = \emptyset \iff A \subset B$$

### Свойства декартова произведения

#### 17. Дистрибутивность произведения

$$(A \cup B) \times C = (A \times C) \cup (B \times C)$$ $$A \times (B \cup C) = (A \times B) \cup (A \times C)$$

$$(A \cap B) \times (C \cap D) = (A \times C) \cap (B \times D)$$

#### 18. Произведение с пустым множеством

$$A \times \emptyset = \emptyset \times A = \emptyset$$

## Примеры

### Применение коммутативности

$${1, 2} \cup {3, 4} = {3, 4} \cup {1, 2} = {1, 2, 3, 4}$$

### Применение ассоциативности

$$({1} \cup {2}) \cup {3} = {1} \cup ({2} \cup {3}) = {1, 2, 3}$$

Можно писать просто ${1} \cup {2} \cup {3}$.

### Применение дистрибутивности

**Раскрытие скобок**: $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

**Конкретный пример**: $${1, 2} \cap ({2, 3} \cup {1, 4})$$ $$= {1, 2} \cap {1, 2, 3, 4}$$ $$= {1, 2}$$

С другой стороны: $$({1, 2} \cap {2, 3}) \cup ({1, 2} \cap {1, 4})$$ $$= {2} \cup {1}$$ $$= {1, 2}$$

### Применение законов поглощения

$$A \cup (A \cap B) = A$$

**Пример**: $${1, 2, 3} \cup ({1, 2, 3} \cap {2, 3, 4})$$ $$= {1, 2, 3} \cup {2, 3}$$ $$= {1, 2, 3}$$

### Доказательство тождества методом включений

**Утверждение**: $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$

**Доказательство**:

1. Покажем $A \cap (B \cup C) \subset (A \cap B) \cup (A \cap C)$:
    
    - Пусть $x \in A \cap (B \cup C)$;
    - Тогда $x \in A$ и $x \in (B \cup C)$;
    - Значит, $x \in A$ и $(x \in B$ или $x \in C)$;
    - Следовательно, $(x \in A \land x \in B)$ или $(x \in A \land x \in C)$;
    - То есть $x \in (A \cap B)$ или $x \in (A \cap C)$;
    - Значит, $x \in (A \cap B) \cup (A \cap C)$.
2. Обратное включение доказывается аналогично

### Упрощение выражений

**Пример 1**: $$A \cup A = A \quad \text{(идемпотентность)}$$

**Пример 2**: $$(A \cup B) \cap A = A \quad \text{(закон поглощения)}$$

**Пример 3**: $$C_M(C_M A) = A \quad \text{(инволютивность)}$$

### Проверка свойств на конкретных множествах

Пусть $M = {1, 2, 3, 4, 5}$, $A = {1, 2}$, $B = {2, 3}$.

**Правило де Моргана**: $$C_M(A \cup B) = C_M {1, 2, 3} = {4, 5}$$ $$C_M A \cap C_M B = {3, 4, 5} \cap {1, 4, 5} = {4, 5}$$
