## 2.2. Матрицы (стр. 41)

### [[Матрица]]

**Связи:**

- [[Сложение матриц]] — покомпонентная операция для матриц одинакового размера
- [[Умножение матриц]] — операция, требующая согласованности размерностей
- [[Умножение на скаляр]] — внешняя операция векторного пространства
- [[Единичная матрица]] — нейтральный элемент умножения
- [[Math/Base/Zettelkasten/Система линейных уравнений]] — матрицы обеспечивают компактную запись $Ax=b$
- [[Math/Core/Zettelkasten/Транспонирование матрицы]] — отражение относительно главной диагонали
- [[Обратная матрица]] — аналог деления (когда существует)

### 2.2.1. Сложение и перемножение матриц (стр. 42)

#### [[Сложение матриц]]

**Связи:**

- [[Матрица]] — сложение является базовой операцией над матрицами
- [[Группа (математическая структура)]] — $(\mathbb{R}^{m \times n}, +)$ образует абелеву группу
- [[Абелева группа]] — сложение матриц коммутативно
- [[Векторное пространство]] — сложение матриц является внутренней операцией векторного пространства
- [[Умножение на скаляр]] — вместе определяют структуру векторного пространства
- [[Math/Core/Zettelkasten/Транспонирование матрицы]] — сложение коммутирует с транспонированием

#### [[Единичная матрица]]

**Связи:**

- [[Матрица]] — единичная матрица является специальным видом квадратной матрицы
- [[Умножение матриц]] — единичная матрица является нейтральным элементом
- [[Обратная матрица]] — связь через $AA^{-1} = I_n = A^{-1}A$
- [[Группа (математическая структура)]] — $I_n$ является нейтральным элементом в группе
- [[Полная линейная группа]] — $I_n \in GL(n, \mathbb{R})$
- [[Math/Core/Zettelkasten/Транспонирование матрицы]] — единичная матрица симметрична: $I_n^\top = I_n$

#### [[Умножение матриц]]

**Связи:**

- [[Матрица]] — базовый объект операции
- [[Math/Base/Zettelkasten/Система линейных уравнений]] — запись $Ax=b$ как умножение матрицы на вектор
- [[Компактное представление системы уравнений]] — произведение $Ax$ как линейная комбинация столбцов
- [[Обратная матрица]] — связь через $AA^{-1}=I$
- [[Линейные отображения]] — матрицы кодируют линейные функции
- [[Матричное представление линейных отображений]] — композиция отображений = умножение матриц

### 2.2.2. Обращение и транспонирование (стр. 44)

#### [[Обратная матрица]]

**Связи:**

- [[Матрица]] — обратная определена только для квадратных матриц
- [[Единичная матрица]] — связь через $AA^{-1} = I$
- [[Math/Base/Zettelkasten/Система линейных уравнений]] — решение $x=A^{-1}b$ при обратимости $A$
- [[Гауссово исключение]] — практический метод вычисления $A^{-1}$
- [[Ранг матрицы]] — критерий обратимости: $\mathrm{rank}(A) = n$
- [[Псевдообратная матрица Мура–Пенроуза]] — обобщение для необратимых и/или неквадратных матриц
- [[Полная линейная группа]] — множество обратимых матриц

#### [[Math/Core/Zettelkasten/Транспонирование матрицы]]

**Связи:**

- [[Матрица]] — базовый объект операции
- [[Симметричная матрица]] — определяется через $A=A^\top$
- [[Умножение матриц]] — при транспонировании произведения меняется порядок
- [[Обратная матрица]] — связь через $(A^{-1})^\top = (A^\top)^{-1}$
- [[Сложение матриц]] — транспонирование дистрибутивно относительно сложения

#### [[Симметричная матрица]]

**Связи:**

- [[Math/Core/Zettelkasten/Транспонирование матрицы]] — симметричность определяется через $A=A^\top$
- [[Матрица]] — свойство определено только для квадратных матриц
- [[Сложение матриц]] — сумма симметричных матриц симметрична
- [[Обратная матрица]] — обратная к симметричной матрице симметрична

### 2.2.3. Умножение на скаляр (стр. 46)

#### [[Умножение на скаляр]]

**Связи:**

- [[Умножение матриц]] — умножение на скаляр коммутирует с умножением матриц
- [[Math/Core/Zettelkasten/Транспонирование матрицы]] — операции коммутируют

## 2.4. Векторные пространства (стр. 58)

### 2.4.1. Группы (стр. 58)

#### [[Группа (математическая структура)]]

**Связи:**

#### [[Абелева группа]]

**Связи:**

- [[Векторное пространство]] — требует абелеву группу по сложению как часть определения

- [[Группа (математическая структура)]] — $GL(n,\mathbb{R})$ удовлетворяет аксиомам группы

- [[Ранг матрицы]] — условие принадлежности: $\mathrm{rank}(A) = n$
- [[Линейные отображения]] — элементы группы представляют обратимые преобразования
