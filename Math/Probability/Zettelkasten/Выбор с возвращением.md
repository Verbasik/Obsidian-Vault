## Суть

**Выбор с возвращением** — это схема выборки, при которой извлеченный элемент возвращается обратно перед следующим извлечением.

При выборе с возвращением:

- На каждом шаге доступны **все M элементов**;
- Элементы могут **повторяться** в выборке;
- Каждое извлечение происходит из **полного** набора.

---

## Детали

### Механизм выборки

**Процесс выбора n элементов с возвращением:**

1. Извлекаем элемент $a_1$ (один из M);
2. **Возвращаем** $a_1$ обратно;
3. Извлекаем элемент $a_2$ (снова один из M);
4. **Возвращаем** $a_2$ обратно;
5. ... (продолжаем n раз).

Результат: последовательность $(a_1, a_2, \ldots, a_n)$ или множество $[a_1, a_2, \ldots, a_n]$

### Ключевые свойства

1. **Повторения разрешены**: $a_i$ может равняться $a_j$ при $i \neq j$;
2. **Независимость извлечений**: результат на шаге i не влияет на возможности на шаге j;
3. **Постоянный выбор**: на каждом шаге M вариантов.

### Число выборок с возвращением

|Тип выборки|Формула|Название|
|---|---|---|
|Упорядоченная|$M^n$|Размещения с повторениями|
|Неупорядоченная|$C^n_{M+n-1} = \binom{M+n-1}{n}$|Сочетания с повторениями|

### Связь с задачей размещения дробинок

В задаче размещения дробинок по ячейкам:

- **Выбор с возвращением** ↔ **Без запрета** (в ячейке может быть любое число дробинок)

Выборка $(a_1, \ldots, a_n)$ означает: дробинка i попадает в ячейку $a_i$

Дробинки могут попасть в одну ячейку → повторения разрешены

### Статистические модели

**С возвращением** соответствуют:

- **Статистика Максвелла-Больцмана**: различимые частицы, без запрета Паули ($M^n$ микросостояний);
- **Статистика Бозе-Эйнштейна**: неразличимые частицы, без запрета ($C^n_{M+n-1}$ микросостояний).

## Примеры

**Пример 1**: Многократное подбрасывание монеты

Подбрасываем монету 5 раз.

Каждый бросок: М = 2 варианта (Г или Р)

Результаты не влияют друг на друга → это выбор с возвращением

Число исходов: $2^5 = 32$

Пример последовательности: $(Г, Г, Р, Г, Г)$ — герб повторяется


**Пример 2**: Выбор шаров из урны с возвращением

Урна с 10 пронумерованными шарами.

Извлекаем шар, записываем номер, **возвращаем**.

Повторяем 3 раза.

Число упорядоченных выборок: $10^3 = 1000$

Возможна выборка $(5, 5, 5)$ — один и тот же шар три раза!


**Пример 3**: Бросок кости n раз

Бросаем кость 4 раза подряд.

M = 6 граней, n = 4 броска

Число последовательностей: $6^4 = 1296$

Возможна $(1, 1, 1, 1)$ — все броски дали 1


**Пример 4**: Выбор букв для пароля

Пароль из 8 букв латинского алфавита (с повторениями).

M = 26 букв, n = 8 позиций

Число паролей: $26^8 \approx 2.08 \times 10^{11}$

Возможен пароль "AAAAAAAA"


**Пример 5**: Размещение частиц по уровням энергии

5 различимых электронов, 3 энергетических уровня.

На одном уровне может быть несколько электронов → с возвращением

Число микросостояний (Максвелл-Больцман): $3^5 = 243$

## Источник

- Документ: [[Вероятность.pdf]]
- Раздел/страницы: § 1, Пример 4, стр. 24; Пример 6, стр. 27
- Ключевая цитата:
    
    > «Выбор с возвращением. Так называют эксперимент, в котором на каждом шаге извлеченный шар возвращается обратно. В этом случае каждая выборка из n шаров может быть записана в виде $(a_1, \ldots, a_n)$, где $a_i$ — номер шара, извлеченного на i-м шаге. Понятно, что в случае выбора с возвращением каждое $a_i$ может принимать любое из M значений $1, 2, \ldots, M$».
