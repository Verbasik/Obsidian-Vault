## Суть

**Статистика Ферми-Дирака** — это комбинаторная модель распределения **неразличимых частиц** по **квантовым состояниям** с **запретом Паули**: не более одной частицы может находиться в каждом состоянии.

Число способов разместить $n$ неразличимых фермионов по $M$ квантовым состояниям:

$$N(\Omega) = C_M^n = \frac{M!}{n!(M-n)!}$$

**Физический смысл**: выбираем $n$ состояний из $M$ для заполнения одной частицей каждое.

---

## Детали

### Определение через задачу размещения дробинок

**Постановка**:

- Имеется $M$ ячеек (квантовых состояний);
- Размещаются $n$ дробинок (частиц);
- **Дробинки неразличимы** (не имеют индивидуальных номеров);
- **С запретом**: в каждой ячейке может быть не более одной дробинки.

**Пространство исходов**: $$\Omega = {[\alpha_1, \ldots, \alpha_n] : \alpha_k \neq \alpha_l \text{ при } k \neq l, ; \alpha_i \in {1,\ldots,M}}$$

где $[\alpha_1, \ldots, \alpha_n]$ — неупорядоченный набор (порядок не важен).

**Число исходов**: $N(\Omega) = C_M^n$

### Связь с другими статистиками

**Сравнение трёх квантовых статистик**:

|Статистика|Частицы|Запрет Паули|Формула|
|---|---|---|---|
|Максвелла-Больцмана|Различимые|Нет|$M^n$|
|Бозе-Эйнштейна|Неразличимые|Нет|$C_{M+n-1}^n$|
|Ферми-Дирака|Неразличимые|Да|$C_M^n$|

**Принцип запрета Паули**: два фермиона не могут находиться в одном квантовом состоянии.

### Физическая интерпретация

**Фермионы** — частицы с полуцелым спином:

- Электроны;
- Протоны;
- Нейтроны;
- Кварки;
- Нейтрино.

**Бозоны** (подчиняются статистике Бозе-Эйнштейна) — частицы с целым спином:

- Фотоны;
- Глюоны;
- W и Z бозоны;
- Пи-мезоны.

**Ограничение**: $n \leq M$ (число частиц не может превышать число состояний)

### Связь с комбинаторикой выборок

Статистика Ферми-Дирака соответствует:

- **Неупорядоченным выборкам без возвращения**;
- Выбор $n$ элементов из $M$ без учёта порядка;
- Каждый элемент может быть выбран не более одного раза.

**Двойственность**:

```
Неупорядоченные выборки    ↔    Неразличимые дробинки
    без возвращения        ↔      с запретом Паули
```

## Примеры

### Пример 1: Заполнение электронных оболочек

Рассмотрим упрощённую модель атома с $M = 4$ орбиталями и $n = 2$ электронами.

**Вопрос**: Сколько способов разместить 2 электрона по 4 орбиталям?

**Решение**: $$N = C_4^2 = \frac{4!}{2! \cdot 2!} = \frac{24}{4} = 6$$

**Визуализация** (• — электрон, ○ — пустая орбиталь):

1. [••○○]
2. [•○•○]
3. [•○○•]
4. [○••○]
5. [○•○•]
6. [○○••]

### Пример 2: Три частицы в трёх ячейках

$M = 3$ ячейки, $n = 3$ фермиона

$$N = C_3^3 = 1$$

Единственная конфигурация: все ячейки заполнены по одной частице.

**Это характерно для фермионов**: при $n = M$ существует ровно одна конфигурация.

### Пример 3: Сравнение статистик для $M=3$, $n=2$

**Максвелла-Больцмана** (различимые, без запрета): $$N_{MB} = M^n = 3^2 = 9$$

Конфигурации: (11), (12), (13), (21), (22), (23), (31), (32), (33)

**Бозе-Эйнштейна** (неразличимые, без запрета): $$N_{BE} = C_{M+n-1}^n = C_4^2 = 6$$

Конфигурации: [11], [12], [13], [22], [23], [33]

**Ферми-Дирака** (неразличимые, с запретом): $$N_{FD} = C_M^n = C_3^2 = 3$$

Конфигурации: [12], [13], [23]

### Пример 4: Вероятность заполнения состояний

Пусть $M = 10$ энергетических уровней, $n = 3$ электрона распределяются случайно (равновероятно по всем конфигурациям).

**Вопрос**: Какова вероятность, что первый уровень занят?

**Решение**:

- Всего конфигураций: $C_{10}^3 = 120$
- Конфигураций с заполненным первым уровнем: выбираем 2 из оставшихся 9 уровней: $C_9^2 = 36$

$$P(\text{первый уровень занят}) = \frac{36}{120} = \frac{3}{10}$$

**Общая формула**: Вероятность заполнения любого конкретного уровня: $$P = \frac{n}{M}$$
