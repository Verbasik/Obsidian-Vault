## Суть

**Статистика Бозе-Эйнштейна** — это способ подсчета микросостояний системы **неразличимых** частиц (бозонов), не подчиняющихся принципу запрета Паули.

Число микросостояний для n неразличимых бозонов на M энергетических уровнях: $$ W_{BE} = C^n_{M+n-1} = \binom{M+n-1}{n} = \frac{(M+n-1)!}{n!(M-1)!} $$

Это соответствует **неупорядоченной выборке с возвращением** (сочетания с повторениями).

---

## Детали

### Условия применимости

Статистика Бозе-Эйнштейна применяется, когда:

1. **Частицы неразличимы** (тождественные частицы);
2. **Без принципа запрета** — на одном уровне может быть произвольное число частиц;
3. **Целочисленный спин** — частицы являются бозонами (спин 0, 1, 2, ...).

### Бозоны

Частицы со **целочисленным спином**, подчиняющиеся статистике Бозе-Эйнштейна:

- **Фотоны** (спин 1) — кванты электромагнитного поля;
- **Пи-мезоны** (спин 0);
- **W- и Z-бозоны** (спин 1) — переносчики слабого взаимодействия;
- **Глюоны** (спин 1) — переносчики сильного взаимодействия;
- **Бозон Хиггса** (спин 0);
- **Атомы с четным числом фермионов** (например, ⁴He).

### Комбинаторная интерпретация

**Задача**: разместить n **неразличимых** дробинок по M ячейкам **без запрета**.

Важно только: сколько дробинок в каждой ячейке — вектор $(k_1, \ldots, k_M)$ с $k_1 + \cdots + k_M = n$ и $k_i \geq 0$.

Число таких векторов: $C^n_{M+n-1}$ (метод "звезды и палочки")

### Физические эффекты

**Бозе-конденсация**: при низких температурах макроскопическое число бозонов переходит в основное состояние ($k_1 \approx n$, остальные $k_i \approx 0$).

Примеры:

- Сверхтекучесть гелия-4;
- Бозе-конденсаты атомов щелочных металлов;
- Когерентность лазерного излучения.

### Сравнение со статистикой Максвелла-Больцмана

Для n частиц на M уровнях:

|Статистика|Формула|Пример (n=3, M=5)|
|---|---|---|
|Максвелла-Больцмана (различимы)|$M^n$|$5^3 = 125$|
|Бозе-Эйнштейна (неразличимы)|$C^n_{M+n-1}$|$C^3_7 = 35$|

Неразличимость **уменьшает** число микросостояний.

## Примеры

**Пример 1**: Фотоны в полости

3 неразличимых фотона в полости с 5 модами колебаний.

Фотоны — бозоны, могут быть в одной моде.

Число микросостояний: $C^3_{5+3-1} = C^3_7 = 35$

Примеры распределений (количество фотонов в каждой моде):

- $(3, 0, 0, 0, 0)$ — все фотоны в моде 1;
- $(1, 1, 1, 0, 0)$ — по одному в первых трех модах;
- $(2, 1, 0, 0, 0)$ — два в первой, один во второй.


**Пример 2**: Пи-мезоны на энергетических уровнях

2 пи-мезона, 4 энергетических уровня.

Число микросостояний: $C^2_{4+2-1} = C^2_5 = 10$


**Пример 3**: Распределение одинаковых шаров

5 одинаковых красных шаров в 3 ящика.

Шары неразличимы.

Число способов: $C^5_{3+5-1} = C^5_7 = 21$

Примеры: $(5,0,0), (4,1,0), (3,2,0), (3,1,1), (2,2,1), \ldots$


**Пример 4**: Неотрицательные решения уравнения

Число неотрицательных целых решений $x_1 + x_2 + x_3 + x_4 = 10$:

Это эквивалентно размещению 10 неразличимых единиц по 4 переменным.

Число решений: $C^{10}_{4+10-1} = C^{10}_{13} = 286$


**Пример 5**: Бозе-конденсат

1000 атомов ⁴He в ловушке с 100 энергетическими состояниями.

При очень низкой температуре практически все атомы в основном состоянии: $(1000, 0, 0, \ldots, 0)$ — **Бозе-конденсат**

Это один из $C^{1000}_{100+1000-1} \approx 10^{215}$ возможных микросостояний!


**Пример 6**: Сравнение трех статистик

n = 3 частицы, M = 5 энергетических уровней:

|Статистика|Различимость|Запрет|Формула|Число|
|---|---|---|---|---|
|Максвелла-Больцмана|Различимы|Нет|$M^n$|125|
|Бозе-Эйнштейна|Неразличимы|Нет|$C^n_{M+n-1}$|35|
|Ферми-Дирака|Неразличимы|Да|$C^n_M$|10|
