## Суть

**Аксиомы вероятности** (аксиомы Колмогорова) — это три основных постулата, которым должна удовлетворять вероятностная мера P.

**Аксиома 1** (Неотрицательность): $P(A) \geq 0$ для всех событий A.

**Аксиома 2** (Нормированность): $P(\Omega) = 1$.

**Аксиома 3** (Аддитивность): Если события A и B несовместны ($A \cap B = \emptyset$), то: $$ P(A \cup B) = P(A) + P(B) $$

Эти аксиомы были сформулированы А. Н. Колмогоровым в 1933 году и стали основой современной теории вероятностей.

---

## Детали

### Аксиома 1: Неотрицательность

**Формулировка**: $$ P(A) \geq 0 \quad \text{для всех } A \in \mathscr{A} $$

**Смысл**: Вероятность не может быть отрицательной.

**Следствие**: Вероятность — это число из отрезка $[0, 1]$.

**Интерпретация**: Отражает то, что «частота появления события» неотрицательна.

### Аксиома 2: Нормированность

**Формулировка**: $$ P(\Omega) = 1 $$

**Смысл**: Достоверное событие (всё пространство) имеет вероятность 1.

**Интерпретация**:

- При любом исходе эксперимента произойдёт **хотя бы одно** элементарное событие;
- Сумма вероятностей всех элементарных событий равна 1.

**Нормировка**: Эта аксиома задаёт масштаб вероятностей (единицу измерения).

### Аксиома 3: Аддитивность

**Формулировка** (для двух событий): $$ \text{Если } A \cap B = \emptyset, \text{ то } P(A \cup B) = P(A) + P(B) $$

**Обобщение** (конечная аддитивность): Для попарно несовместных событий $A_1, \ldots, A_n$: $$ P\left(\bigcup_{i=1}^n A_i\right) = \sum_{i=1}^n P(A_i) $$

**Смысл**: Вероятности несовместных событий складываются.

**Интерпретация**: Если события не могут произойти одновременно, вероятность «хотя бы одного» равна сумме вероятностей.

### Расширение: σ-аддитивность

Для **счётного** числа попарно несовместных событий $A_1, A_2, \ldots$: $$ P\left(\bigcup_{i=1}^\infty A_i\right) = \sum_{i=1}^\infty P(A_i) $$

Это свойство необходимо для работы с бесконечными пространствами и непрерывными распределениями.

В элементарной теории (конечное Ω) достаточно конечной аддитивности.

### Минимальность аксиом

Эти три аксиомы:

- **Минимальны**: из них выводятся все остальные свойства вероятности;
- **Независимы**: ни одна не следует из других двух;
- **Достаточны**: определяют понятие вероятностной меры полностью.

### Следствия из аксиом

Из трёх аксиом выводятся:

1. $P(\emptyset) = 0$;
2. $P(\bar{A}) = 1 - P(A)$;
3. Если $A \subseteq B$, то $P(A) \leq P(B)$;
4. $P(A \cup B) = P(A) + P(B) - P(A \cap B)$;
5. $P(A) \leq 1$ для всех A.

## Примеры

**Пример 1**: Проверка аксиом для честной монеты

$\Omega = {Г, Р}$, $P({Г}) = P({Р}) = 1/2$

**Аксиома 1**: $P({Г}) = 1/2 \geq 0$ ✓, $P({Р}) = 1/2 \geq 0$ ✓

**Аксиома 2**: $P(\Omega) = P({Г, Р}) = P({Г}) + P({Р}) = 1/2 + 1/2 = 1$ ✓

**Аксиома 3**: События ${Г}$ и ${Р}$ несовместны. $P({Г} \cup {Р}) = P(\Omega) = 1 = 1/2 + 1/2 = P({Г}) + P({Р})$ ✓


**Пример 2**: Бросок кости

$\Omega = {1, 2, 3, 4, 5, 6}$, $P({i}) = 1/6$ для всех i

$A = {2, 4, 6}$ (четные), $B = {1, 3, 5}$ (нечетные)

События A и B несовместны: $A \cap B = \emptyset$

По аксиоме 3: $$ P(A \cup B) = P(\Omega) = 1 = P(A) + P(B) = 3/6 + 3/6 = 1 $$ 

**Пример 3**: Вывод $P(\emptyset) = 0$ из аксиом

Используем аксиому 3 с $A = \Omega$ и $B = \emptyset$:

$\Omega$ и $\emptyset$ несовместны (формально: $\Omega \cap \emptyset = \emptyset$)

По аксиоме 3: $$ P(\Omega \cup \emptyset) = P(\Omega) + P(\emptyset) $$

Но $\Omega \cup \emptyset = \Omega$, поэтому: $$ P(\Omega) = P(\Omega) + P(\emptyset) $$

Отсюда $P(\emptyset) = 0$ ✓


**Пример 4**: Вывод формулы дополнения

Для любого события A: $\Omega = A \cup \bar{A}$ и $A \cap \bar{A} = \emptyset$

По аксиоме 3: $$ P(\Omega) = P(A \cup \bar{A}) = P(A) + P(\bar{A}) $$

По аксиоме 2: $P(\Omega) = 1$

Следовательно: $$ 1 = P(A) + P(\bar{A}) \quad \Rightarrow \quad P(\bar{A}) = 1 - P(A) $$

**Пример 5**: Аддитивность для трёх событий

$A_1 = {1, 2}$, $A_2 = {3, 4}$, $A_3 = {5, 6}$ — попарно несовместны

$P(A_i) = 2/6 = 1/3$ для всех i

По аксиоме 3: $$ P(A_1 \cup A_2 \cup A_3) = P(A_1) + P(A_2) + P(A_3) = 1/3 + 1/3 + 1/3 = 1 $$

Что совпадает с $P(\Omega) = 1$ ✓


**Пример 6**: Нарушение аксиом

Рассмотрим функцию $Q: \mathscr{A} \to \mathbb{R}$, где $Q({Г}) = 0.6$, $Q({Р}) = 0.6$

Проверим аксиому 2: $$ Q(\Omega) = Q({Г, Р}) = Q({Г}) + Q({Р}) = 0.6 + 0.6 = 1.2 \neq 1 $$

Аксиома 2 нарушена! Значит, Q **не является** вероятностной мерой.

- Раздел/страницы: § 1, п. 4, стр. 32-33
- Ключевая цитата (определение вероятности):
    
    > «Припишем каждому элементарному событию (исходу, явлению) $\omega_i \in \Omega$, i = 1, ..., N, некоторый «вес», обозначаемый $p(\omega_i)$ (или $p_i$) и называемый вероятностью исхода $\omega_i$, который будем считать удовлетворяющим следующим условиям: a) $0 \leq p(\omega_i) \leq 1$ (неотрицательность), b) $p(\omega_1) + ... + p(\omega_N) = 1$ (нормированность)».

- Можно ли заменить аксиому аддитивности на другую эквивалентную?
- Как аксиомы связаны с частотной интерпретацией вероятности?
- Что изменится, если отказаться от нормированности ($P(\Omega) = 1$)?
