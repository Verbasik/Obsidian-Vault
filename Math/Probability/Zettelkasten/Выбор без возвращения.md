## Суть

**Выбор без возвращения** — это схема выборки, при которой извлеченный элемент **не возвращается** обратно.

При выборе без возвращения:

- Элементы **не могут повторяться** в выборке;
- С каждым извлечением число доступных элементов **уменьшается**;
- Требуется $n \leq M$ (нельзя извлечь больше, чем есть).

---

## Детали

### Механизм выборки

**Процесс выбора n элементов без возвращения:**

1. Извлекаем элемент $a_1$ (один из M элементов);
2. **Не возвращаем** $a_1$;
3. Извлекаем элемент $a_2$ (один из оставшихся M-1);
4. **Не возвращаем** $a_2$;
5. ... (продолжаем n раз).

Результат: последовательность $(a_1, a_2, \ldots, a_n)$ или множество $[a_1, a_2, \ldots, a_n]$, где все элементы **различны**: $a_i \neq a_j$ при $i \neq j$.

### Ключевые свойства

1. **Повторения запрещены**: $a_i \neq a_j$ при $i \neq j$;
2. **Зависимость извлечений**: результат на шаге i влияет на возможности на шаге j;
3. **Уменьшающийся выбор**: на шаге i доступно M-(i-1) вариантов;
4. **Ограничение**: необходимо $n \leq M$.

### Число выборок без возвращения

|Тип выборки|Формула|Название|
|---|---|---|
|Упорядоченная|$(M)_n = M(M-1)\cdots(M-n+1) = \frac{M!}{(M-n)!}$|Размещения без повторений|
|Неупорядоченная|$C^n_M = \binom{M}{n} = \frac{M!}{n!(M-n)!}$|Сочетания без повторений|

### Связь с задачей размещения дробинок

В задаче размещения дробинок по ячейкам:

- **Выбор без возвращения** ↔ **С запретом** (в каждой ячейке не более одной дробинки)

Требуется $n \leq M$ (дробинок не больше, чем ячеек)

### Статистическая модель

**Без возвращения** соответствует:

- **Статистика Ферми-Дирака**: неразличимые частицы, принцип запрета Паули ($C^n_M$ микросостояний)

Применяется к фермионам: электронам, протонам, нейтронам.

## Примеры

**Пример 1**: Лотерея

Из барабана с 49 шарами извлекают 6 шаров (без возвращения).

Порядок не важен → неупорядоченная выборка.

Число комбинаций: $C^6_{49} = \frac{49!}{6! \cdot 43!} = 13,983,816$

Каждый шар может быть выбран не более одного раза.


**Пример 2**: Раздача карт

Из колоды 52 карты раздаются 4 игрокам по 13 карт.

Каждая карта используется ровно один раз → без возвращения.

Число способов раздать первому игроку: $C^{13}_{52}$


**Пример 3**: Выбор комитета

Из 10 человек выбирают комитет из 3 человек.

Один человек не может войти в комитет дважды → без возвращения.

Порядок не важен: $C^3_{10} = \frac{10!}{3! \cdot 7!} = 120$


**Пример 4**: Финиш в забеге

8 бегунов финишируют (порядок важен).

Каждый финиширует ровно один раз → без возвращения.

Число возможных результатов: $8! = 40,320$


**Пример 5**: Выбор 2 шаров из урны с 5 шарами

**Упорядоченная выборка**: Число способов: $(5)_2 = 5 \cdot 4 = 20$

Примеры: $(1,2), (1,3), (1,4), (1,5), (2,1), \ldots$

Но **нет** $(1,1)$, $(2,2)$, и т.д. — повторения запрещены!

**Неупорядоченная выборка**: Число способов: $C^2_5 = \frac{5!}{2! \cdot 3!} = 10$

Примеры: $[1,2], [1,3], [1,4], [1,5], [2,3], \ldots$


**Пример 6**: Размещение электронов

3 электрона на 5 энергетических уровнях.

Принцип Паули: на одном уровне не более одного электрона → без возвращения.

Электроны неразличимы: $C^3_5 = 10$ микросостояний (Ферми-Дирак)
