## Суть

**Разбиение пространства** Ω — это представление Ω в виде объединения попарно непересекающихся непустых подмножеств.

Система множеств $\mathscr{D} = {D_1, D_2, \ldots, D_n}$ образует разбиение Ω, если:

1. **Непустота**: $D_i \neq \emptyset$ для всех i;
2. **Непересекаемость**: $D_i \cap D_j = \emptyset$ при $i \neq j$;
3. **Покрытие**: $D_1 \cup D_2 \cup \cdots \cup D_n = \Omega$.

Множества $D_i$ называются **атомами разбиения**.

---

## Детали

### Определение разбиения

Система непустых множеств $\mathscr{D} = {D_1, \ldots, D_n}$ называется **разбиением** (или **партицией**) множества Ω, если:

$$ D_i \cap D_j = \emptyset \quad (i \neq j), \quad \bigcup_{i=1}^n D_i = \Omega $$

Это означает:

- Каждый элемент $\omega \in \Omega$ принадлежит ровно одному атому $D_i$;
- Атомы **исчерпывают** всё пространство без пропусков и пересечений.

### Свойства разбиения

1. **Однозначность**: Каждое $\omega \in \Omega$ принадлежит ровно одному $D_i$;
2. **Полная система событий**: Атомы разбиения образуют полную систему несовместных событий;
3. **Сумма вероятностей**: $$ \sum_{i=1}^n P(D_i) = P(\Omega) = 1 $$
4. **Индикаторное представление**: Для каждого $\omega$ ровно один индикатор $\mathbb{1}_{D_i}(\omega) = 1$, остальные равны 0.

### Число разбиений

Для множества из N элементов число различных разбиений равно **числу Белла** $B_N$: $$ B_N = e^{-1} \sum_{k=0}^\infty \frac{k^N}{k!} $$

Примеры:

- $B_1 = 1$ (только ${{1}}$);
- $B_2 = 2$ (разбиения: ${{1}, {2}}$ и ${{1, 2}}$);
- $B_3 = 5$;
- $B_4 = 15$ю

### Алгебра, порожденная разбиением

Разбиение $\mathscr{D}$ порождает алгебру $\alpha(\mathscr{D})$, состоящую из:

- Пустого множества $\emptyset$;
- Всевозможных объединений атомов.

Число элементов алгебры: $|\alpha(\mathscr{D})| = 2^n$

### Отношение «мельче»

Разбиение $\mathscr{D}_2$ **мельче** разбиения $\mathscr{D}_1$ (обозначение: $\mathscr{D}_1 \preccurlyeq \mathscr{D}_2$), если: $$ \alpha(\mathscr{D}_1) \subseteq \alpha(\mathscr{D}_2) $$

То есть каждый атом $\mathscr{D}_1$ представим как объединение атомов $\mathscr{D}_2$.

### Тривиальные разбиения

**Самое грубое**: $\mathscr{D}_1 = {\Omega}$ (один атом — всё пространство)

**Самое мелкое**: $\mathscr{D}_{\max} = {{\omega_1}, {\omega_2}, \ldots, {\omega_N}}$ (каждый элемент — отдельный атом)

## Примеры

**Пример 1**: Разбиение множества из 3 элементов

$\Omega = {1, 2, 3}$

Все возможные разбиения (число Белла $B_3 = 5$):

1. $\mathscr{D}_1 = {{1, 2, 3}}$ — одна часть;
2. $\mathscr{D}_2 = {{1, 2}, {3}}$;
3. $\mathscr{D}_3 = {{1, 3}, {2}}$;
4. $\mathscr{D}_4 = {{2, 3}, {1}}$;
5. $\mathscr{D}_5 = {{1}, {2}, {3}}$ — три части.


**Пример 2**: Бросок кости по четности

$\Omega = {1, 2, 3, 4, 5, 6}$

Разбиение: $D_1 = {2, 4, 6}$ (четные), $D_2 = {1, 3, 5}$ (нечетные)

$\mathscr{D} = {D_1, D_2}$

Проверка:

- $D_1 \cap D_2 = \emptyset$ ✓
- $D_1 \cup D_2 = \Omega$ ✓

Алгебра $\alpha(\mathscr{D}) = {\emptyset, D_1, D_2, \Omega}$ содержит 4 элемента.


**Пример 3**: Двукратное подбрасывание монеты по числу гербов

$\Omega = {ГГ, ГР, РГ, РР}$

Разбиение по числу гербов:

- $D_0 = {РР}$ — 0 гербов;
- $D_1 = {ГР, РГ}$ — 1 герб;
- $D_2 = {ГГ}$ — 2 герба.

$\mathscr{D} = {D_0, D_1, D_2}$

Атомы непересекаются и покрывают Ω.

$P(D_0) = 1/4$, $P(D_1) = 1/2$, $P(D_2) = 1/4$

Сумма: $1/4 + 1/2 + 1/4 = 1$ ✓


**Пример 4**: Разбиение по интервалам

$\Omega = [0, 10]$ (непрерывное пространство — для иллюстрации)

Разбиение: $D_1 = [0, 5)$, $D_2 = [5, 10]$

Атомы покрывают $[0, 10]$ без пропусков и пересечений.


**Пример 5**: Сравнение разбиений

$\Omega = {1, 2, 3, 4}$

$\mathscr{D}_1 = {{1, 2}, {3, 4}}$ — грубое разбиение (2 атома)

$\mathscr{D}_2 = {{1}, {2}, {3, 4}}$ — более мелкое (3 атома)

$\mathscr{D}_2$ мельче $\mathscr{D}_1$: $\mathscr{D}_1 \preccurlyeq \mathscr{D}_2$

Действительно: атом ${1, 2}$ из $\mathscr{D}_1$ представим как ${1} \cup {2}$ из $\mathscr{D}_2$.


**Пример 6**: Событие и его дополнение

Для любого события $A \subseteq \Omega$ пара ${A, \bar{A}}$ образует разбиение Ω:

- $A \cap \bar{A} = \emptyset$
- $A \cup \bar{A} = \Omega$

Это простейшее нетривиальное разбиение.

    > «Будем говорить, что система множеств $\mathscr{D} = {D_1, \ldots, D_n}$ образует разбиение множества Ω, а $D_i$ являются атомами этого разбиения, если множества $D_i$ непусты, попарно не пересекаются и их сумма равна Ω: $D_1 + \ldots + D_n = \Omega$».
