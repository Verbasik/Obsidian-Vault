#### [[Алгебра, порожденная разбиением]]

### 1.5. Вероятностное пространство (стр. 32–33)

#### [[Вероятность исхода]]

**Связи:**

- [[Элементарное событие]] — каждому ωᵢ приписывается p(ωᵢ)
- [[Вероятность события]] — P(A) = Σ_{ωᵢ∈A} p(ωᵢ)
- [[Вероятностная мера]] — вероятностная функция ( $P : \mathcal A \to [0,1]$ ), удовлетворяющая аксиомам Колмогорова

#### [[Неотрицательность вероятности]]

**Связи:**

- [[Вероятность исхода]] — 0 ≤ p(ωᵢ) ≤ 1
- [[Аксиомы вероятности]] — первая аксиома
- [[Вероятность события]] — 0 ≤ P(A) ≤ 1

#### [[Нормировка вероятности]]

**Связи:**

- [[Вероятность исхода]] — Σp(ωᵢ) = 1
- [[Аксиомы вероятности]] — вторая аксиома (нормированность)
- [[Достоверное событие]] — P(Ω) = 1

#### [[Вероятность события]]

**Связи:**

- [[Событие]] — каждому A ∈ A приписывается P(A)
- [[Вероятность исхода]] — P(A) = Σ_{ωᵢ∈A} p(ωᵢ)
- [[Аксиомы вероятности]] — определяющие свойства
- [[Свойства вероятности]] — следствия из аксиом
- [[Вероятностное пространство]] — P — третий компонент (Ω, A, P)

#### [[Вероятностное пространство]]

**Связи:**

- [[Пространство элементарных событий]] — первый компонент Ω
- [[Алгебра событий]] — второй компонент A
- [[Вероятность события]] — третий компонент P
- [[Вероятностная модель]] — формализация эксперимента

#### [[Вероятностная модель]]

**Связи:**

- [[Вероятностное пространство]] — математическая реализация модели
- [[Эксперимент (испытание)]] — модель описывает эксперимент

#### [[Аксиомы вероятности]]

**Связи:**

- [[Аддитивность вероятности]] — P(A + B) = P(A) + P(B)
- [[Свойства вероятности]] — следствия из аксиом

#### [[Свойства вероятности]]

**Связи:**

- [[Аксиомы вероятности]] — выводятся из аксиом
- [[Формула включений-исключений]] — P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
- [[Монотонность вероятности]] — если A ⊆ B, то P(A) ≤ P(B)
