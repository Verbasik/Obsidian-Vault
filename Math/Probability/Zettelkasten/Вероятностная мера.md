## Суть

**Вероятностная мера** (или просто **вероятность**) — это функция P, которая каждому событию A из алгебры $\mathscr{A}$ сопоставляет число $P(A) \in [0, 1]$, удовлетворяющее аксиомам вероятности.

Вероятностная мера определяется как функция: $$ P: \mathscr{A} \to [0, 1] $$

удовлетворяющая:

1. **Неотрицательность**: $P(A) \geq 0$ для всех $A \in \mathscr{A}$;
2. **Нормированность**: $P(\Omega) = 1$;
3. **Аддитивность**: Для несовместных событий $P(A \cup B) = P(A) + P(B)$.

---

## Детали

### Аксиоматическое определение

Пусть $(\Omega, \mathscr{A})$ — измеримое пространство (пространство элементарных событий с алгеброй событий).

Функция $P: \mathscr{A} \to [0, 1]$ называется **вероятностной мерой**, если выполнены аксиомы Колмогорова:

**Аксиома 1** (Неотрицательность): $$ P(A) \geq 0 \quad \text{для всех } A \in \mathscr{A} $$

**Аксиома 2** (Нормированность): $$ P(\Omega) = 1 $$

**Аксиома 3** (Аддитивность для несовместных событий): Если $A \cap B = \emptyset$, то: $$ P(A \cup B) = P(A) + P(B) $$

Для конечного числа попарно несовместных событий $A_1, \ldots, A_n$: $$ P\left(\bigcup_{i=1}^n A_i\right) = \sum_{i=1}^n P(A_i) $$

### Конструкция вероятностной меры

В элементарной теории (конечное Ω) вероятностная мера задается через вероятности элементарных событий:

1. Каждому $\omega_i \in \Omega$ приписываем вес $p(\omega_i) \geq 0$;
2. Требуем $\sum_{i=1}^N p(\omega_i) = 1$;
3. Для любого события $A \subseteq \Omega$ определяем: $$ P(A) = \sum_{\omega_i \in A} p(\omega_i) $$

### Интерпретации вероятности

**Частотная интерпретация**: $$ P(A) = \lim_{n \to \infty} \frac{\text{число появлений A в n испытаниях}}{n} $$

**Классическая** (равновозможные исходы): $$ P(A) = \frac{|A|}{|\Omega|} $$

**Субъективная** (степень уверенности, байесовский подход)

### Свойства вероятностной меры

1. **Вероятность невозможного события**: $P(\emptyset) = 0$;
2. **Вероятность дополнения**: $P(\bar{A}) = 1 - P(A)$;
3. **Монотонность**: Если $A \subseteq B$, то $P(A) \leq P(B)$;
4. **Формула включений-исключений**: $$ P(A \cup B) = P(A) + P(B) - P(A \cap B) $$
5. **Неравенство Буля**: $$ P\left(\bigcup_{i=1}^n A_i\right) \leq \sum_{i=1}^n P(A_i) $$

### Мера vs Вероятность

**Мера** — общее понятие из теории меры (может быть $\mu(\Omega) = \infty$)

**Вероятностная мера** — специальный случай с $P(\Omega) = 1$ (нормированная мера)

## Примеры

**Пример 1**: Равномерная вероятность на монете

$\Omega = {Г, Р}$

$p(Г) = p(Р) = 1/2$

Для события $A = {Г}$: $$ P(A) = p(Г) = 1/2 $$

Проверка аксиом:

- $P(A) = 1/2 \geq 0$ ✓
- $P(\Omega) = p(Г) + p(Р) = 1/2 + 1/2 = 1$ ✓
- События ${Г}$ и ${Р}$ несовместны: $P({Г} \cup {Р}) = 1 = 1/2 + 1/2$ ✓


**Пример 2**: Несимметричная монета

$\Omega = {Г, Р}$

$p(Г) = 0.7$, $p(Р) = 0.3$

$P({Г}) = 0.7$, $P({Р}) = 0.3$

$P(\Omega) = 0.7 + 0.3 = 1$ ✓


**Пример 3**: Бросок кости

$\Omega = {1, 2, 3, 4, 5, 6}$

Честная кость: $p(i) = 1/6$ для всех $i$

Событие $A = {2, 4, 6}$ (четное число): $$ P(A) = p(2) + p(4) + p(6) = 1/6 + 1/6 + 1/6 = 1/2 $$

**Пример 4**: Дискретное распределение

$\Omega = {1, 2, 3}$

$p(1) = 1/2$, $p(2) = 1/3$, $p(3) = 1/6$

Проверка: $1/2 + 1/3 + 1/6 = 3/6 + 2/6 + 1/6 = 1$ ✓

$P({1, 2}) = 1/2 + 1/3 = 5/6$

$P({1}) = 1/2$


**Пример 5**: Вычисление через дополнение

Подбрасываем монету 3 раза.

$A$ = «хотя бы один герб»

$\bar{A}$ = «ни одного герба» = ${РРР}$

$P(\bar{A}) = (1/2)^3 = 1/8$

$P(A) = 1 - P(\bar{A}) = 1 - 1/8 = 7/8$


**Пример 6**: Непрерывное пространство (для иллюстрации)

$\Omega = [0, 1]$

Равномерная вероятность на отрезке: $$ P([a, b]) = b - a \quad (0 \leq a \leq b \leq 1) $$

Проверка: $P([0, 1]) = 1 - 0 = 1$ ✓

$P([0, 0.5]) = 0.5$

## Вопросы

- Почему аксиома нормированности ($P(\Omega) = 1$) важна для интерпретации?
- Можно ли задать вероятностную меру, не задавая вероятности элементарных событий?
- Как обобщается аддитивность на счётное число событий (σ-аддитивность)?
- Всегда ли вероятность можно интерпретировать как предел частот?
