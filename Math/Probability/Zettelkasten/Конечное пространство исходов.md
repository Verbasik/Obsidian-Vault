## Суть

**Конечное пространство исходов** — это пространство элементарных событий Ω, содержащее **конечное** число элементов:

$$ \Omega = {\omega_1, \omega_2, \ldots, \omega_N}, \quad N < \infty $$

где N — общее число различных исходов эксперимента.

Элементарная теория вероятностей занимается исключительно случаем конечных пространств.

---

## Детали

### Элементарная теория вероятностей

**Элементарная теория** — это часть теории вероятностей, в которой:

- Пространство Ω конечно: |Ω| = N < ∞;
- Приходится иметь дело с вероятностями **конечного** числа событий;
- Все вероятности вычисляются через конечные суммы.

Такое ограничение позволяет:

- Избежать тонких вопросов теории меры;
- Использовать комбинаторные методы;
- Получать явные формулы для вероятностей.

### Мощность пространства

Обозначения для числа элементов Ω:

- N(Ω) — число исходов;
- |Ω| — мощность множества (кардинальность);
- просто N, когда контекст ясен.

### Преимущества конечности

**1. Простота вычислений**

- Вероятность события: P(A) = Σ_{ωᵢ∈A} p(ωᵢ) — конечная сумма;
- Не нужны интегралы и пределы;
- Все операции сводятся к арифметике.

**2. Применимость комбинаторики**

- Подсчет числа исходов N(A) комбинаторными методами;
- Классическое определение вероятности: P(A) = N(A)/N;
- Широкий класс практических задач.

**3. Алгебра событий**

- Можно рассматривать все подмножества Ω;
- Алгебра событий A = 2^Ω содержит 2^N элементов;
- Не возникает парадоксов теории множеств.

### Ограничения конечности

Конечные пространства не покрывают:

- Непрерывные измерения (время, расстояние, вес);
- Счетное число испытаний (бесконечная последовательность бросков монеты);
- Геометрические вероятности.

Для таких задач требуется **общая теория вероятностей** с бесконечными пространствами.

## Примеры

**Пример 1**: Однократное подбрасывание монеты $$ \Omega = {\text{Г}, \text{Р}}, \quad N = 2 $$

**Пример 2**: n-кратное подбрасывание монеты $$ \Omega = {(a_1, \ldots, a_n) : a_i \in {\text{Г}, \text{Р}}}, \quad N = 2^n $$

**Пример 3**: Бросок игральной кости $$ \Omega = {1, 2, 3, 4, 5, 6}, \quad N = 6 $$

**Пример 4**: Выбор 2 шаров из урны с M шарами без возвращения (неупорядоченная выборка) $$ \Omega = {[a_1, a_2] : 1 \leq a_1 < a_2 \leq M}, \quad N = C^2_M = \frac{M(M-1)}{2} $$

**Пример 5**: Эксперимент "монета, затем кость или монета" (стр. 24 книги) $$ \Omega = {\text{Г}1, \text{Г}2, \text{Г}3, \text{Г}4, \text{Г}5, \text{Г}6, \text{РГ}, \text{РР}}, \quad N = 8 $$

**Пример 6**: Размещение n различимых дробинок по M ячейкам $$ N = M^n $$ Пример: 5 частиц по 3 уровням → N = 3^5 = 243
