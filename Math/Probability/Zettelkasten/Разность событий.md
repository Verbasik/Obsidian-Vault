## Суть

**Разность событий** A \ B (или A - B) — это событие, которое происходит тогда и только тогда, когда происходит A, но не происходит B.

Формально: $$ A \setminus B = {\omega \in \Omega : \omega \in A \text{ и } \omega \notin B} = A \cap \bar{B} $$

где $\bar{B}$ — дополнение события B.

---

## Детали

### Определение

**Разность** A \ B состоит из всех элементарных событий, которые входят в A, но не входят в B.

### Обозначения

- $A \setminus B$ (основное);
- $A - B$;
- $A \cap \bar{B}$ (через дополнение).

### Связь с другими операциями

**Через дополнение и пересечение**: $$ A \setminus B = A \cap \bar{B} $$

**Симметричная разность**: $$ A \triangle B = (A \setminus B) \cup (B \setminus A) $$

(события происходят, но не одновременно)

### Свойства разности

**Неком мутативность**: $A \setminus B \neq B \setminus A$ в общем случае;

**Ассоциативность НЕ выполняется**: $(A \setminus B) \setminus C \neq A \setminus (B \setminus C)$;

**Вычитание пустого множества**: $A \setminus \emptyset = A$;

**Вычитание из себя**: $A \setminus A = \emptyset$;

**Вычитание из пространства**: $\Omega \setminus A = \bar{A}$ (дополнение);

**Вычитание дополнения**: $A \setminus \bar{B} = A \cap B$.

### Формула для вероятности

$$ P(A \setminus B) = P(A \cap \bar{B}) = P(A) - P(A \cap B) $$

**Частный случай** (если $B \subseteq A$): $$ P(A \setminus B) = P(A) - P(B) $$

### Диаграммы Эйлера-Венна

На диаграмме A \ B — это область A, из которой вырезана область B.

```
     ┌─────────────┐
     │      A      │
     │  ┌───────┐  │
     │  │ A∩B   │  │──▶ эта часть удаляется
     │  └───────┘  │
     │   A \ B     │──▶ эта часть остаётся
     └─────────────┘
```

## Примеры

**Пример 1**: Бросок кости

$A = {1, 2, 3, 4}$ (число ≤ 4)

$B = {2, 4, 6}$ (чётное)

$A \setminus B = {1, 3}$ (нечётное И ≤ 4)

Проверка: $A \cap \bar{B} = {1,2,3,4} \cap {1,3,5} = {1,3}$ ✓


**Пример 2**: Колода карт

$A$ = «красная карта» (26 карт)

$B$ = «фигура» (король, дама, валет; 12 карт)

$A \setminus B$ = «красная карта, но не фигура» = красные карты от 2 до 10 плюс тузы червей и бубен

Число карт: $26 - 6 = 20$


**Пример 3**: Вероятность

Бросок кости: $P(A) = 4/6 = 2/3$, $P(B) = 3/6 = 1/2$

$A \cap B = {2, 4}$, $P(A \cap B) = 2/6 = 1/3$

$$ P(A \setminus B) = P(A) - P(A \cap B) = \frac{2}{3} - \frac{1}{3} = \frac{1}{3} $$

Проверка: $A \setminus B = {1, 3}$, $P = 2/6 = 1/3$ ✓


**Пример 4**: Вложенные события

$A = {1, 2, 3, 4, 5}$

$B = {2, 3} \subset A$ (B вложено в A)

$A \setminus B = {1, 4, 5}$

$P(A \setminus B) = P(A) - P(B) = 5/6 - 2/6 = 3/6 = 1/2$


**Пример 5**: Непересекающиеся события

$A = {1, 2}$, $B = {5, 6}$

$A \cap B = \emptyset$

$A \setminus B = A = {1, 2}$

$P(A \setminus B) = P(A) = 2/6 = 1/3$


**Пример 6**: Симметричная разность

Бросок монеты дважды:

$A$ = «первый герб» = ${ГГ, ГР}$

$B$ = «второй герб» = ${ГГ, РГ}$

$A \setminus B = {ГР}$ (первый Г, второй Р)

$B \setminus A = {РГ}$ (первый Р, второй Г)

$A \triangle B = {ГР, РГ}$ (ровно один герб)

## Источник

- Документ: [[Вероятность.pdf]]
- Раздел/страницы: § 1, п. 3, стр. 29-30
- Ключевая цитата:
    
    > «Если через $B \setminus A$ обозначать разность множеств B и A (т. е. множество точек, входящих в B и не входящих в A), то $\bar{A} = \Omega \setminus A$».
