## Суть

**Алгебра событий** — это система подмножеств пространства элементарных событий Ω, замкнутая относительно операций объединения, пересечения и дополнения.

Алгебра событий $\mathscr{A}$ — это семейство подмножеств Ω, удовлетворяющее аксиомам:

1. $\Omega \in \mathscr{A}$;
2. Если $A \in \mathscr{A}$, $B \in \mathscr{A}$, то $A \cup B, A \cap B, A \setminus B \in \mathscr{A}$.

В элементарной теории вероятностей обычно рассматривают **все** подмножества Ω.

---

## Детали

### Аксиомы алгебры

Система подмножеств $\mathscr{A} \subseteq 2^\Omega$ называется **алгеброй**, если:

1. **Содержит пространство**: $\Omega \in \mathscr{A}$;

2. **Замкнутость относительно операций**:

    - Если $A, B \in \mathscr{A}$, то $A \cup B \in \mathscr{A}$ (объединение);
    - Если $A, B \in \mathscr{A}$, то $A \cap B \in \mathscr{A}$ (пересечение);
    - Если $A, B \in \mathscr{A}$, то $A \setminus B \in \mathscr{A}$ (разность).

### Следствия из аксиом

Из аксиом следует:

1. **Пустое множество**: $\emptyset \in \mathscr{A}$ (так как $\emptyset = \Omega \setminus \Omega$);
2. **Дополнение**: Если $A \in \mathscr{A}$, то $\bar{A} \in \mathscr{A}$ (так как $\bar{A} = \Omega \setminus A$);
3. **Конечные объединения**: Если $A_1, \ldots, A_n \in \mathscr{A}$, то $\bigcup_{i=1}^n A_i \in \mathscr{A}$;
4. **Конечные пересечения**: Если $A_1, \ldots, A_n \in \mathscr{A}$, то $\bigcap_{i=1}^n A_i \in \mathscr{A}$.

### Минимальная алгебра

**Тривиальная алгебра**: $\mathscr{A} = {\Omega, \emptyset}$

Это минимальная алгебра, содержащая только достоверное и невозможное события.

### Максимальная алгебра

**Алгебра всех подмножеств**: $\mathscr{A} = 2^\Omega$ (степенное множество)

Содержит **все** возможные подмножества Ω.

Для конечного пространства $|\Omega| = N$ имеем $|\mathscr{A}| = 2^N$.

### Алгебра, порожденная событием

Для события $A \subseteq \Omega$ минимальная алгебра, содержащая A: $$ \mathscr{A} = {A, \bar{A}, \Omega, \emptyset} $$

Содержит 4 элемента.

### Алгебра, порожденная разбиением

Пусть $\mathscr{D} = {D_1, D_2, \ldots, D_n}$ — разбиение Ω (атомы попарно не пересекаются и покрывают Ω).

Алгебра $\alpha(\mathscr{D})$, порожденная разбиением, состоит из:

- Пустого множества $\emptyset$;
- Всевозможных объединений атомов $D_i$.

Число элементов: $2^n$ (для каждого атома: включать или не включать в объединение)

### Роль алгебры событий

Алгебра событий определяет:

- **Какие подмножества** Ω мы называем событиями;
- На каких множествах **определена вероятность** P;
- **Структуру** пространства событий.

## Примеры

**Пример 1**: Тривиальная алгебра

$\Omega = {1, 2, 3}$

$\mathscr{A} = {\emptyset, \Omega} = {\emptyset, {1, 2, 3}}$

Содержит только 2 элемента.


**Пример 2**: Алгебра, порожденная одним событием

$\Omega = {1, 2, 3, 4}$

$A = {1, 2}$

$\mathscr{A} = {\emptyset, {1,2}, {3,4}, {1,2,3,4}}$

Содержит 4 элемента: $\emptyset$, $A$, $\bar{A}$, $\Omega$.


**Пример 3**: Алгебра всех подмножеств

$\Omega = {1, 2}$

$\mathscr{A} = {\emptyset, {1}, {2}, {1,2}}$

Содержит $2^2 = 4$ элемента — все возможные подмножества.


**Пример 4**: Алгебра, порожденная разбиением

$\Omega = {1, 2, 3, 4, 5, 6}$

Разбиение: $D_1 = {1, 2}$, $D_2 = {3, 4}$, $D_3 = {5, 6}$

Алгебра $\alpha(\mathscr{D})$ содержит:

- $\emptyset$;
- $D_1, D_2, D_3$ (одиночные атомы);
- $D_1 \cup D_2 = {1,2,3,4}$, $D_1 \cup D_3 = {1,2,5,6}$, $D_2 \cup D_3 = {3,4,5,6}$ (пары);
- $D_1 \cup D_2 \cup D_3 = \Omega$ (все).

Всего: $2^3 = 8$ элементов


**Пример 5**: Бросок монеты

$\Omega = {Г, Р}$

Алгебра всех событий: $$ \mathscr{A} = {\emptyset, {Г}, {Р}, {Г, Р}} $$

События:

- $\emptyset$ — «невозможное»;
- ${Г}$ — «выпал герб»;
- ${Р}$ — «выпала решка»;
- $\Omega$ — «достоверное».

**Пример 6**: Двукратное подбрасывание

$\Omega = {ГГ, ГР, РГ, РР}$, $|\Omega| = 4$

Алгебра всех подмножеств содержит $2^4 = 16$ событий.

Примеры событий:

- $A = {ГГ, ГР}$ — «первый бросок — герб»
- $B = {ГГ, РГ}$ — «второй бросок — герб»
- $A \cup B = {ГГ, ГР, РГ}$ — «хотя бы один герб»
- $A \cap B = {ГГ}$ — «оба герба»

Все эти множества принадлежат алгебре $\mathscr{A}$.

## Источник

- Документ: [[Вероятность.pdf]]
- Раздел/страницы: § 1, п. 3, стр. 30-31
- Ключевая цитата:
    
    > «Если рассматривается некоторая система $\mathscr{A}_0$ множеств $A \subseteq \Omega$, то с помощью теоретико-множественных операций $\cup$, $\cap$ и $\setminus$ можно из элементов $\mathscr{A}_0$ построить новую систему множеств, которые также являются событиями. Присоединяя к этим событиям достоверное и невозможное события Ω и $\emptyset$, получаем систему множеств $\mathscr{A}$, которая является алгеброй».
