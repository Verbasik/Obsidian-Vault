## Суть

**Различимые объекты** — это объекты, которые можно отличить друг от друга по некоторому признаку (номер, цвет, метка и т.д.).

В комбинаторике и теории вероятностей:

- Различимые объекты → **упорядоченные** выборки;
- Порядок размещения объектов имеет значение;
- Перестановка объектов дает новую конфигурацию.

---

## Детали

### Примеры различимости

Объекты различимы, если:

- **Пронумерованы**: шары с номерами 1, 2, 3, ...;
- **Различного цвета**: белая, черная, красная дробинки;
- **Имеют имена**: Алиса, Боб, Карл;
- **Физически различны**: разные электроны в атоме (у каждого свой набор квантовых чисел).

### Соответствие с выборками

В задаче **выбора шаров из урны**:

- **Различимые объекты** ↔ **Упорядоченные выборки**;
- Выборка $(a_1, \ldots, a_n)$ показывает: на шаге i выбран объект $a_i$.

В задаче **размещения дробинок по ячейкам**:

- **Различимые дробинки** ↔ **Упорядоченные выборки**;
- Выборка $(a_1, \ldots, a_n)$ показывает: дробинка №i попала в ячейку $a_i$.

### Число конфигураций с различимыми объектами

|Условие|Формула|
|---|---|
|Без запрета (с возвращением)|$M^n$|
|С запретом (без возвращения)|$(M)_n = M!/(M-n)!$|

где M — число ячеек, n — число различимых объектов.

### Пример: различимые vs неразличимые

**Задача**: разместить 2 дробинки по 3 ячейкам.

**Различимые дробинки** (белая Б и черная Ч):

Конфигурации (используем таблицу):

```
Ячейка 1  Ячейка 2  Ячейка 3
   БЧ         —          —
   Б          Ч          —
   Б          —          Ч
   Ч          Б          —
   —         БЧ          —
   —          Б          Ч
   Ч          —          Б
   —          Ч          Б
   —          —         БЧ
```

Всего: $3^2 = 9$ конфигураций

**Неразличимые дробинки** (обе одинаковые ++):

Конфигурации:

```
Ячейка 1  Ячейка 2  Ячейка 3
   ++         —          —
   +          +          —
   +          —          +
   —         ++          —
   —          +          +
   —          —         ++
```

Всего: $C^2_{3+2-1} = C^2_4 = 6$ конфигураций

Различимость **увеличивает** число возможных конфигураций!

## Примеры

**Пример 1**: Нумерованные шары

Урна содержит 5 шаров с номерами 1, 2, 3, 4, 5.

Шары различимы по номерам.

Извлекаем 3 шара без возвращения (порядок важен).

Число способов: $(5)_3 = 5 \cdot 4 \cdot 3 = 60$


**Пример 2**: Цветные дробинки

Размещаем белую, черную и красную дробинки по 4 ячейкам.

Дробинки различимы по цвету.

Число способов: $4^3 = 64$

Пример: (Б в ячейке 1, Ч в ячейке 1, К в ячейке 2) — одна конфигурация


**Пример 3**: Именованные люди

5 человек (Алиса, Боб, Карл, Диана, Ева) садятся на 3 стула.

Люди различимы по именам.

Порядок важен (стулья пронумерованы).

Число способов: $(5)_3 = 60$


**Пример 4**: Статистика Максвелла-Больцмана

n **различимых** частиц (классические частицы с координатами) распределяются по M энергетическим уровням.

Без принципа запрета (на уровне может быть любое число частиц).

Число микросостояний: $M^n$


**Пример 5**: Раздача различимых карт

Раздаем 5 различимых карт (туз пик, король червей, ...) двум игрокам.

Карты различимы.

Число способов: $2^5 = 32$

Каждая карта может пойти либо первому, либо второму игроку.
