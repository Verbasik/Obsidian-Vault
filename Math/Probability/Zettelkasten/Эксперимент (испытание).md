## Суть

**Эксперимент (испытание)** — это процесс наблюдения или измерения, результаты которого описываются конечным числом различных исходов (элементарных событий).

Эксперимент проводится при определенном **комплексе условий**, который определяет:

- Какие исходы возможны;
- Какие характеристики результата регистрируются;
- Какие внешние факторы учитываются.

---

## Детали

### Компоненты эксперимента

**1. Комплекс условий**

- Описывает условия проведения эксперимента;
- Определяет множество возможных исходов;
- Обычно «по умолчанию» предполагается данным;
- Разные комплексы условий → разные вероятностные модели.

**2. Множество исходов Ω**

- Все мыслимые результаты эксперимента;
- Определяется комплексом условий;
- Исходы должны быть взаимоисключающими.

**3. Наблюдаемые характеристики**

- Что именно регистрируется в результате эксперимента;
- Определяет уровень детализации пространства Ω.

### Воспроизводимость эксперимента

Эксперимент должен быть **воспроизводимым**:

- Можно повторить при тех же условиях;
- Результат может меняться от опыта к опыту;
- Именно случайность результата делает эксперимент предметом теории вероятностей.

### Случайность vs детерминизм

- **Детерминированный эксперимент**: результат однозначно определяется начальными условиями;
- **Случайный эксперимент**: результат не определяется однозначно, но подчиняется вероятностным закономерностям.

В теории вероятностей изучаются случайные эксперименты.

### Роль комплекса условий

Комплекс условий важен, так как для одного физического действия возможны разные вероятностные модели:

**Пример**: Трехкратное подбрасывание монеты

_Комплекс условий 1_: Регистрируем все три результата

- Ω = {ГГГ, ГГР, ГРГ, РГГ, ГРР, РГР, РРГ, РРР}
- |Ω| = 8

_Комплекс условий 2_: Регистрируем только первый бросок

- Ω = {Г, Р}
- |Ω| = 2

_Комплекс условий 3_: Регистрируем только общее число гербов

- Ω = {0, 1, 2, 3}
- |Ω| = 4

## Примеры

**Пример 1**: Подбрасывание монеты

- Эксперимент: подброс монеты и наблюдение результата;
- Комплекс условий: монета может упасть только на одну из двух сторон;
- Исходы: Ω = {Г, Р}.

**Пример 2**: Бросок игральной кости

- Эксперимент: бросок кости и фиксация выпавшего числа;
- Комплекс условий: кость правильной формы, одна из шести граней оказывается сверху;
- Исходы: Ω = {1, 2, 3, 4, 5, 6}.

**Пример 3**: Выбор шара из урны

- Эксперимент: извлечение шара и определение его номера;
- Комплекс условий: урна содержит M пронумерованных шаров, извлекается один;
- Исходы: Ω = {1, 2, ..., M}.

**Пример 4**: Измерение времени реакции

- Эксперимент: предъявление стимула и измерение времени ответа;
- Комплекс условий: время измеряется с точностью до миллисекунды;
- Исходы: Ω = {0, 1, 2, ..., T} (дискретизированное время).
