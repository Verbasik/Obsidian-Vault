## Суть

**Законы Моргана** (De Morgan) — это два фундаментальных правила, связывающие операции объединения, пересечения и дополнения для событий:

$$ \overline{A \cup B} = \bar{A} \cap \bar{B} $$

$$ \overline{A \cap B} = \bar{A} \cup \bar{B} $$

**Словами**:

- Дополнение объединения = пересечение дополнений;
- Дополнение пересечения = объединение дополнений.

---

## Детали

### Формулировка

**Первый закон**: $$ \overline{A \cup B} = \bar{A} \cap \bar{B} $$

«Не (A или B)» = «(не A) и (не B)»

**Второй закон**: $$ \overline{A \cap B} = \bar{A} \cup \bar{B} $$

«Не (A и B)» = «(не A) или (не B)»

### Обобщение на n событий

Для произвольного числа событий $A_1, \ldots, A_n$:

**Первый закон**: $$ \overline{\bigcup_{i=1}^n A_i} = \bigcap_{i=1}^n \bar{A}_i $$

**Второй закон**: $$ \overline{\bigcap_{i=1}^n A_i} = \bigcup_{i=1}^n \bar{A}_i $$

### Доказательство (для двух событий)

**Первый закон**: $\overline{A \cup B} = \bar{A} \cap \bar{B}$

$\omega \in \overline{A \cup B}$ ⟺ $\omega \notin A \cup B$ ⟺ $\omega \notin A$ и $\omega \notin B$ ⟺ $\omega \in \bar{A}$ и $\omega \in \bar{B}$ ⟺ $\omega \in \bar{A} \cap \bar{B}$ ✓

**Второй закон**: $\overline{A \cap B} = \bar{A} \cup \bar{B}$

$\omega \in \overline{A \cap B}$ ⟺ $\omega \notin A \cap B$ ⟺ не ($\omega \in A$ и $\omega \in B$) ⟺ $\omega \notin A$ или $\omega \notin B$ ⟺ $\omega \in \bar{A}$ или $\omega \in \bar{B}$ ⟺ $\omega \in \bar{A} \cup \bar{B}$ ✓

### Связь с логикой

Законы Моргана соответствуют правилам логики:

**Логика**:

- $\neg (A \lor B) = (\neg A) \land (\neg B)$;
- $\neg (A \land B) = (\neg A) \lor (\neg B)$.

**Теория множеств**:

- $\overline{A \cup B} = \bar{A} \cap \bar{B}$;
- $\overline{A \cap B} = \bar{A} \cup \bar{B}$.

где $\neg$ (не), $\lor$ (или), $\land$ (и).

### Применение к вероятностям

Из законов Моргана следуют формулы для вероятностей:

$$ P(\overline{A \cup B}) = P(\bar{A} \cap \bar{B}) $$

$$ P(\overline{A \cap B}) = P(\bar{A} \cup \bar{B}) $$

Полезно для вычислений через дополнения.

## Примеры

**Пример 1**: Бросок кости

$A = {1, 2, 3}$ (число ≤ 3)

$B = {2, 4, 6}$ (чётное)

**Применяем первый закон**:

$A \cup B = {1, 2, 3, 4, 6}$

$\overline{A \cup B} = {5}$

$\bar{A} = {4, 5, 6}$, $\bar{B} = {1, 3, 5}$

$\bar{A} \cap \bar{B} = {5}$ ✓

**Применяем второй закон**:

$A \cap B = {2}$

$\overline{A \cap B} = {1, 3, 4, 5, 6}$

$\bar{A} \cup \bar{B} = {4, 5, 6} \cup {1, 3, 5} = {1, 3, 4, 5, 6}$ ✓


**Пример 2**: Подбрасывание монеты дважды

$A$ = «первый герб» = ${ГГ, ГР}$

$B$ = «второй герб» = ${ГГ, РГ}$

**Первый закон**:

$A \cup B$ = «хотя бы один герб» = ${ГГ, ГР, РГ}$

$\overline{A \cup B}$ = «ни одного герба» = ${РР}$

$\bar{A} = {РГ, РР}$ (первый не герб)

$\bar{B} = {ГР, РР}$ (второй не герб)

$\bar{A} \cap \bar{B} = {РР}$ ✓

**Второй закон**:

$A \cap B$ = «оба герба» = ${ГГ}$

$\overline{A \cap B} = {ГР, РГ, РР}$

$\bar{A} \cup \bar{B} = {РГ, РР} \cup {ГР, РР} = {ГР, РГ, РР}$ ✓


**Пример 3**: Три события

$\Omega = {1, 2, 3, 4, 5, 6}$

$A_1 = {1, 2}$, $A_2 = {3, 4}$, $A_3 = {5, 6}$

$A_1 \cup A_2 \cup A_3 = \Omega$

$\overline{A_1 \cup A_2 \cup A_3} = \bar{\Omega} = \emptyset$

$\bar{A}_1 \cap \bar{A}_2 \cap \bar{A}_3 = {3,4,5,6} \cap {1,2,5,6} \cap {1,2,3,4} = \emptyset$ ✓


**Пример 4**: Вычисление вероятности

Бросок кости:

$A = {1, 2, 3}$, $P(A) = 1/2$

$B = {2, 4, 6}$, $P(B) = 1/2$

$A \cup B = {1, 2, 3, 4, 6}$, $P(A \cup B) = 5/6$

**Вычислим через закон Моргана**:

$P(\overline{A \cup B}) = P(\bar{A} \cap \bar{B})$

$P(\overline{A \cup B}) = 1 - P(A \cup B) = 1 - 5/6 = 1/6$

$\bar{A} = {4, 5, 6}$, $\bar{B} = {1, 3, 5}$

$\bar{A} \cap \bar{B} = {5}$, $P(\bar{A} \cap \bar{B}) = 1/6$ ✓


**Пример 5**: Логическое отрицание

«Не (студент или преподаватель)» = «Не студент И не преподаватель»
