## Суть

**Классическая вероятность** — это способ задания вероятности, основанный на предположении о **равновозможности** всех исходов эксперимента.

Для конечного пространства элементарных событий $\Omega = {\omega_1, \ldots, \omega_N}$ вероятность события A вычисляется по формуле:

$$ \mathbb{P}(A) = \frac{N(A)}{N(\Omega)} = \frac{\text{число благоприятных исходов}}{\text{общее число исходов}} $$

где:

- $N(A)$ — число элементарных событий, составляющих A;
- $N(\Omega) = N$ — общее число исходов.

---

## Детали

### Предпосылки классического определения

Классическое определение применимо, когда:

1. **Конечность пространства**: $|\Omega| = N < \infty$;
2. **Равновозможность исходов**: все элементарные события имеют одинаковую вероятность $$p(\omega_1) = p(\omega_2) = \ldots = p(\omega_N) = \frac{1}{N}$$
3. **Симметрия/однородность**: нет оснований считать один исход более вероятным, чем другой.

### Обоснование равновозможности

Равновозможность исходов обычно обосновывается через:

- **Физическую симметрию**: симметричная монета, кость правильной формы;
- **Однородность**: шары в урне неразличимы на ощупь;
- **Отсутствие преимуществ**: нет причин для предпочтения одного исхода другому.

### Связь с комбинаторикой

Классическое определение сводит вычисление вероятности к **комбинаторной задаче**:

- Подсчитать общее число исходов N(Ω);
- Подсчитать число благоприятных исходов N(A);
- Вычислить отношение N(A)/N(Ω).

Для подсчета используются формулы:

- Размещения: с повторениями $M^n$, без повторений $(M)_n$;
- Сочетания: с повторениями $C^n_{M+n-1}$, без повторений $C^n_M$;
- Перестановки: $n!$

### Ограничения классического определения

**Недостатки**:

1. Применимо только для конечных пространств;
2. Требует равновозможности исходов;
3. Не всегда очевидно, как выбрать Ω для равновозможности;
4. Циркулярность: «равновозможные» = «имеющие одинаковую вероятность».

**Выход за рамки**:

- Несимметричная монета: P(Г) ≠ P(Р);
- Бесконечное пространство: Ω = ℝ;
- Неоднородные условия.

### Свойства классической вероятности

Все аксиомы вероятности выполняются автоматически:

**Неотрицательность**: $$0 \leq N(A) \leq N(\Omega) \Rightarrow 0 \leq \mathbb{P}(A) \leq 1$$

**Нормированность**: $$\mathbb{P}(\Omega) = \frac{N(\Omega)}{N(\Omega)} = 1$$

**Аддитивность**: если $A \cap B = \emptyset$, то $N(A + B) = N(A) + N(B)$, следовательно $$\mathbb{P}(A + B) = \frac{N(A) + N(B)}{N(\Omega)} = \mathbb{P}(A) + \mathbb{P}(B)$$

### Историческая роль

Классическое определение:

- Исторически первое строгое определение вероятности;
- Доминировало в XVIII-XIX веках;
- Основа для развития комбинаторных методов;
- Стимулировало развитие азартных игр и их анализа;
- Предшественник современной аксиоматики Колмогорова (1933).

## Примеры

**Пример 1**: Вероятность выпадения четного числа на симметричной кости

$$\Omega = {1, 2, 3, 4, 5, 6}, \quad N(\Omega) = 6$$ $$A = {2, 4, 6}, \quad N(A) = 3$$ $$\mathbb{P}(A) = \frac{3}{6} = \frac{1}{2}$$

**Пример 2**: Вероятность получить хотя бы один герб при трех подбрасываниях монеты

$$\Omega = {\text{ГГГ, ГГР, ГРГ, РГГ, ГРР, РГР, РРГ, РРР}}, \quad N(\Omega) = 8$$ $$A = \Omega \setminus {\text{РРР}}, \quad N(A) = 7$$ $$\mathbb{P}(A) = \frac{7}{8}$$

Или через дополнение: $$\mathbb{P}(A) = 1 - \mathbb{P}({\text{РРР}}) = 1 - \frac{1}{8} = \frac{7}{8}$$

**Пример 3**: Задача о днях рождения (парадокс дней рождения)

В классе n учеников. Какова вероятность, что хотя бы у двух совпадают дни рождения?

Модель: M = 365 дней, выбор с возвращением, упорядоченная выборка

$$N(\Omega) = 365^n$$

Событие $A^c$ = «все дни рождения различны»: $$N(A^c) = (365)_n = 365 \cdot 364 \cdot \ldots \cdot (365 - n + 1)$$

Вероятность совпадения: $$\mathbb{P}(A) = 1 - \frac{(365)_n}{365^n}$$

Для n = 23: $\mathbb{P}(A) \approx 0.507$ (более 50%!)

**Пример 4**: Задача о выигрыше в лотерею

M билетов, из них n выигрышных. Покупаем n билетов. Вероятность хотя бы одного выигрыша?

Модель: неупорядоченная выборка без возвращения

$$N(\Omega) = C^n_M$$

Событие $A^c$ = «все билеты проигрышные»: $$N(A^c) = C^n_{M-n}$$

Вероятность выигрыша: $$\mathbb{P}(A) = 1 - \frac{C^n_{M-n}}{C^n_M}$$

Для M = n²: при n → ∞ имеем $\mathbb{P}(A) \to 1 - e^{-1} \approx 0.632$

## Источник

- Документ: [[Вероятность.pdf]]
- Раздел/страницы: § 1, п. 5, стр. 33–35
- Ключевая цитата:
    
    > «Существует много практических ситуаций, в которых из соображений симметрии или однородности представляется разумным рассматривать все мыслимые исходы как равновозможные. Поэтому, если пространство элементарных исходов Ω состоит из точек ω₁, ..., ωₙ, где N < ∞, то полагают p(ω₁) = ... = p(ωₙ) = 1/N, и, следовательно, для любого события A ∈ A: P(A) = N(A)/N».
