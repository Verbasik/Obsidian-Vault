## Суть

**Распределение Бернулли** — это распределение случайной величины, принимающей два значения: 1 («успех») с вероятностью p и 0 («неудача») с вероятностью q = 1-p.

Обозначение: $X \sim \text{Bern}(p)$

Вероятности: $$ P(X = 1) = p, \quad P(X = 0) = 1-p = q $$

где $0 \leq p \leq 1$.

---

## Детали

### Определение

Случайная величина X имеет **распределение Бернулли** с параметром p, если: $$ P(X = k) = \begin{cases} p, & k = 1 \ 1-p, & k = 0 \ 0, & \text{иначе} \end{cases} $$

**Компактная запись**: $$ P(X = k) = p^k (1-p)^{1-k}, \quad k \in {0, 1} $$

### Параметры

**Параметр**: $p \in [0, 1]$ — вероятность успеха

**Область значений**: ${0, 1}$

### Характеристики

**Математическое ожидание**: $$ E[X] = 0 \cdot (1-p) + 1 \cdot p = p $$

**Дисперсия**: $$ \text{Var}(X) = E[X^2] - (E[X])^2 = p - p^2 = p(1-p) = pq $$

**Максимум дисперсии**: при $p = 1/2$ дисперсия максимальна: $\text{Var}(X) = 1/4$

### Связь с другими распределениями

**Биномиальное**: Сумма n независимых испытаний Бернулли имеет биномиальное распределение: $$ \text{Если } X_i \sim \text{Bern}(p), \text{ то } \sum_{i=1}^n X_i \sim \text{Bin}(n, p) $$

**Геометрическое**: Число испытаний Бернулли до первого успеха имеет геометрическое распределение.

**Индикатор события**: Распределение Бернулли — это распределение индикатора события: $$ X = \mathbb{1}_A, \quad P(A) = p $$

### Примеры испытаний Бернулли

1. **Подбрасывание монеты**: успех = герб, $p = 1/2$ (честная монета);
2. **Стрельба по мишени**: успех = попадание;
3. **Проверка качества**: успех = годная деталь;
4. **Медицинский тест**: успех = выздоровление;
5. **Бинарная классификация**: успех = правильная классификация.

## Примеры

**Пример 1**: Подбрасывание честной монеты

X = 1, если герб; X = 0, если решка

$X \sim \text{Bern}(1/2)$

$P(X = 1) = 1/2$, $P(X = 0) = 1/2$

$E[X] = 1/2$, $\text{Var}(X) = 1/4$


**Пример 2**: Несимметричная монета

Монета с вероятностью герба $p = 0.7$

$X \sim \text{Bern}(0.7)$

$P(X = 1) = 0.7$, $P(X = 0) = 0.3$

$E[X] = 0.7$, $\text{Var}(X) = 0.7 \cdot 0.3 = 0.21$


**Пример 3**: Индикатор события

Бросок кости. Событие A = «выпало четное».

$X = \mathbb{1}_A$ — индикатор события A

$X = 1$, если выпало четное (успех)

$X = 0$, если выпало нечетное (неудача)

$p = P(A) = 1/2$

$X \sim \text{Bern}(1/2)$


**Пример 4**: Стрельба

Стрелок стреляет один раз. Вероятность попадания $p = 0.8$.

$X \sim \text{Bern}(0.8)$

$P(\text{попал}) = 0.8$, $P(\text{промах}) = 0.2$


**Пример 5**: Три независимых испытания Бернулли

Подбрасываем монету 3 раза.

$X_1, X_2, X_3 \sim \text{Bern}(1/2)$ независимо

Каждый $X_i$ принимает значения 0 или 1.

Сумма $S = X_1 + X_2 + X_3$ — число гербов — имеет биномиальное распределение $\text{Bin}(3, 1/2)$.

$P(S = k) = C_3^k (1/2)^3$

- $P(S = 0) = 1/8$
- $P(S = 1) = 3/8$
- $P(S = 2) = 3/8$
- $P(S = 3) = 1/8$
