## Суть

**Сочетания с повторениями** — это неупорядоченные выборки объема n из M элементов с возвращением.

Число различных сочетаний с повторениями из M элементов по n равно: $$ C^n_{M+n-1} = \binom{M+n-1}{n} = \frac{(M+n-1)!}{n!(M-1)!} $$

Обозначения: $C^n_{M+n-1}$, $\bar{C}^n_M$, или $\left(!!{M \choose n}!!\right)$

---

## Детали

### Определение через выборки

При выборе n шаров из урны с M шарами **с возвращением**, **неупорядоченная** выборка записывается как: $$ \omega = [a_1, a_2, \ldots, a_n] $$

где $a_i \in {1, 2, \ldots, M}$, повторения разрешены, порядок не важен.

Для удобства элементы можно записать в неубывающем порядке: $a_1 \leq a_2 \leq \cdots \leq a_n$.

### Вывод формулы (метод "звезды и палочки")

Представим выборку как распределение n **неразличимых шаров** по M **ячейкам**.

Закодируем это как последовательность из n звездочек $\star$ и (M-1) разделителей $|$:

Пример для M=3, n=4: $\star\star|\star|\star$ означает [1,1,2,3]

- В ячейке 1: две звездочки;
- В ячейке 2: одна звездочка;
- В ячейке 3: одна звездочка.

Всего символов: $n + (M-1) = M+n-1$

Число способов выбрать n позиций для звездочек из M+n-1 позиций: $$ C^n_{M+n-1} $$

### Альтернативная формула

Можно также записать через выбор позиций для разделителей: $$ C^{M-1}_{M+n-1} = C^n_{M+n-1} $$

### Рекуррентное соотношение

Формула доказывается по индукции (стр. 25 книги): $$ C^n_{M+n-1} = C^n_{M+n-2} + C^n_{M-1+n-1} + \cdots + C^n_n $$

используя свойство биномиальных коэффициентов: $$ C^{l-1}_k + C^l_k = C^l_{k+1} $$

### Связь со статистикой Бозе-Эйнштейна

Сочетания с повторениями соответствуют **статистике Бозе-Эйнштейна**:

- n **неразличимых** частиц;
- M энергетических уровней (ячеек);
- Без принципа запрета (частицы могут быть на одном уровне).

Применяется к бозонам: фотонам, пи-мезонам.

## Примеры

**Пример 1**: Покупка фруктов

В магазине 4 вида фруктов: яблоки (Я), груши (Г), апельсины (А), бананы (Б).

Покупаем 6 фруктов (с повторениями, порядок не важен).

M = 4, n = 6

Число способов: $C^6_{4+6-1} = C^6_9 = \frac{9!}{6!3!} = 84$

Пример покупки: $[Я, Я, Я, Г, А, Б]$ или $[Б, Б, Б, Б, Б, Б]$ (все бананы!)


**Пример 2**: Уравнение с неотрицательными решениями

Найти число неотрицательных целых решений: $$ x_1 + x_2 + x_3 = 5 $$

Это эквивалентно распределению 5 одинаковых шаров по 3 ячейкам.

M = 3, n = 5

Число решений: $C^5_{3+5-1} = C^5_7 = \frac{7!}{5!2!} = 21$

Примеры: $(5,0,0), (2,2,1), (1,1,3), \ldots$


**Пример 3**: Выбор шаров с возвращением (неупорядоченная выборка)

Из урны с 3 шарами выбираем 2 шара с возвращением, порядок не важен.

M = 3, n = 2

Число выборок: $C^2_{3+2-1} = C^2_4 = 6$

Выборки: $[1,1], [1,2], [1,3], [2,2], [2,3], [3,3]$

Сравните с упорядоченными: $(1,1), (1,2), (1,3), (2,1), (2,2), (2,3), (3,1), (3,2), (3,3)$ — 9 выборок


**Пример 4**: Мономы степени n от M переменных

Число мономов степени n от M переменных $x_1, \ldots, x_M$:

Это эквивалентно распределению степени n между переменными.

Пример для M=3, n=4: $x_1^4, x_1^3x_2, x_1^2x_2^2, x_1^2x_2x_3, \ldots$

Число мономов: $C^n_{M+n-1}$

Для M=3, n=4: $C^4_6 = 15$ мономов


**Пример 5**: Размещение фотонов по энергетическим уровням

3 неразличимых фотона, 5 энергетических уровней.

Фотоны — бозоны, могут быть на одном уровне → с повторениями.

Фотоны неразличимы → неупорядоченная выборка.

Число микросостояний (Бозе-Эйнштейн): $C^3_{5+3-1} = C^3_7 = 35$

- Раздел/страницы: § 1, Пример 4, стр. 24-25; Таблица 4, стр. 28
- Ключевая цитата:
    
    > «Если же рассматриваются неупорядоченные выборки с возвращением (в комбинаторике — сочетания из M по n с повторениями), то Ω = {ω: ω = [a₁, ..., aₙ], aᵢ = 1, ..., M}. Понятно, что число N(Ω) (различных) неупорядоченных выборок меньше, чем число упорядоченных. Покажем, что для этого случая N(Ω) = C^n_{M+n-1}».
