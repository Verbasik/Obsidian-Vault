## Суть

**Аддитивность вероятности** — это фундаментальное свойство, согласно которому вероятность объединения несовместных событий равна сумме их вероятностей.

Для несовместных событий A и B ($A \cap B = \emptyset$): $$ P(A \cup B) = P(A) + P(B) $$

Это одна из аксиом Колмогорова (аксиома 3).

---

## Детали

### Определение

Пусть A и B — два несовместных события: $A \cap B = \emptyset$.

Тогда: $$ P(A \cup B) = P(A) + P(B) $$

**Обозначение суммы**: Для несовместных событий часто пишут $A + B$ вместо $A \cup B$: $$ P(A + B) = P(A) + P(B) $$

### Конечная аддитивность

Для попарно несовместных событий $A_1, A_2, \ldots, A_n$ (то есть $A_i \cap A_j = \emptyset$ при $i \neq j$): $$ P\left(\bigcup_{i=1}^n A_i\right) = \sum_{i=1}^n P(A_i) $$

или в обозначениях суммы: $$ P(A_1 + A_2 + \cdots + A_n) = P(A_1) + P(A_2) + \cdots + P(A_n) $$

### σ-аддитивность (счётная аддитивность)

Для **счётного** числа попарно несовместных событий $A_1, A_2, \ldots$: $$ P\left(\bigcup_{i=1}^\infty A_i\right) = \sum_{i=1}^\infty P(A_i) $$

Это свойство необходимо в общей теории вероятностей для работы с бесконечными пространствами.

В элементарной теории (конечное Ω) достаточно конечной аддитивности.

### Геометрическая интерпретация

Аддитивность означает, что вероятности несовместных событий «не перекрываются» и просто складываются, как площади непересекающихся фигур.

Если события **пересекаются**, то простое сложение даст переоценку (посчитаем пересечение дважды).

### Связь с другими свойствами

**Вытекает из аддитивности**:

1. $P(\emptyset) = 0$;
2. $P(\bar{A}) = 1 - P(A)$;
3. Если $A \subseteq B$, то $P(B \setminus A) = P(B) - P(A)$.

**Не вытекает**:

- Формула включений-исключений для пересекающихся событий (требует дополнительного вывода)

### Практическое применение

Аддитивность позволяет:

- Разбивать сложные события на простые несовместные части;
- Вычислять вероятности через полные системы событий;
- Использовать разбиения пространства для вычислений.

## Примеры

**Пример 1**: Бросок кости

$A = {2, 4}$ (выпало 2 или 4)

$B = {5}$ (выпало 5)

События несовместны: $A \cap B = \emptyset$

$P(A) = 2/6 = 1/3$, $P(B) = 1/6$

$P(A \cup B) = P(A) + P(B) = 1/3 + 1/6 = 1/2$

Проверка: $A \cup B = {2, 4, 5}$, действительно $P({2, 4, 5}) = 3/6 = 1/2$ ✓


**Пример 2**: Полная система событий

Бросок кости. Разбиение по четности:

- $A_1 = {2, 4, 6}$ (четные);
- $A_2 = {1, 3, 5}$ (нечетные).

События несовместны: $A_1 \cap A_2 = \emptyset$

$P(A_1) = 1/2$, $P(A_2) = 1/2$

$P(A_1 \cup A_2) = P(\Omega) = 1 = 1/2 + 1/2 = P(A_1) + P(A_2)$ ✓


**Пример 3**: Трёхкратное подбрасывание монеты

События по числу гербов:

- $A_0 = {РРР}$ — 0 гербов;
- $A_1 = {ГРР, РГР, РРГ}$ — 1 герб;
- $A_2 = {ГГР, ГРГ, РГГ}$ — 2 герба;
- $A_3 = {ГГГ}$ — 3 герба.

События попарно несовместны.

$P(A_0) = 1/8$, $P(A_1) = 3/8$, $P(A_2) = 3/8$, $P(A_3) = 1/8$

По аддитивности: $$ P(A_0 \cup A_1 \cup A_2 \cup A_3) = P(A_0) + P(A_1) + P(A_2) + P(A_3) = 1/8 + 3/8 + 3/8 + 1/8 = 1 $$

**Пример 4**: Вычисление через разбиение

Бросок кости. Найти вероятность $P(A)$, где $A = {1, 3, 5}$ (нечетное).

Разобьем на элементарные события: $$ A = {1} \cup {3} \cup {5} $$

События ${1}$, ${3}$, ${5}$ попарно несовместны.

По аддитивности: $$ P(A) = P({1}) + P({3}) + P({5}) = 1/6 + 1/6 + 1/6 = 1/2 $$

**Пример 5**: Когда аддитивность НЕ применима

Бросок кости.

$A = {1, 2, 3}$ (малое число)

$B = {2, 4, 6}$ (четное)

События **пересекаются**: $A \cap B = {2} \neq \emptyset$

Аддитивность НЕ применима!

$P(A) = 1/2$, $P(B) = 1/2$

$P(A) + P(B) = 1$ — но это **не** равно $P(A \cup B)$!

$A \cup B = {1, 2, 3, 4, 6}$, $P(A \cup B) = 5/6 \neq 1$

Нужна формула включений-исключений: $$ P(A \cup B) = P(A) + P(B) - P(A \cap B) = 1/2 + 1/2 - 1/6 = 5/6 $$ 

**Пример 6**: Вероятность через дополнение (использует аддитивность)

Событие A и его дополнение $\bar{A}$ несовместны: $A \cap \bar{A} = \emptyset$

По аддитивности: $$ P(A \cup \bar{A}) = P(A) + P(\bar{A}) $$

Но $A \cup \bar{A} = \Omega$, поэтому: $$ 1 = P(\Omega) = P(A) + P(\bar{A}) $$

Откуда: $$ P(\bar{A}) = 1 - P(A) $$
