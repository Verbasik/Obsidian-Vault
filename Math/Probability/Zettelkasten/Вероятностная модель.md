## Суть

**Вероятностная модель** — это математическое описание случайного эксперимента, задаваемое тройкой $(\Omega, \mathscr{A}, P)$:

- Ω — пространство элементарных событий;
- A — алгебра событий;
- P — вероятностная мера.

Модель формализует понятие случайности и позволяет вычислять вероятности событий.

---

## Детали

### Определение

**Вероятностная модель** (или **вероятностная теория**) эксперимента — это **вероятностное пространство** $(\Omega, \mathscr{A}, P)$, где:

1. **Ω** — множество всех возможных исходов (пространство элементарных событий);
2. **A** — система подмножеств Ω (алгебра событий);
3. **P** — функция P: A → [0,1] (вероятностная мера), удовлетворяющая аксиомам.

### Компоненты модели

**1. Пространство исходов Ω**:

- Определяется комплексом условий эксперимента;
- Конечное, счётное или несчётное;
- Содержит все мыслимые результаты.

**2. Алгебра событий A**:

- Система подмножеств Ω;
- Замкнута относительно операций ∪, ∩, дополнения;
- В элементарной теории: все подмножества Ω.

**3. Вероятностная мера P**:

- Определяется на A;
- Удовлетворяет аксиомам Колмогорова;
- В конечном случае: задаётся через $p(\omega_i)$.

### Построение модели

**Шаг 1**: Определить пространство Ω

- Описать все возможные исходы;
- Учесть комплекс условий.

**Шаг 2**: Выбрать алгебру A

- Определить, какие события наблюдаемы;
- В простых случаях: все подмножества.

**Шаг 3**: Задать вероятности

- Классический способ (равновозможность);
- Из частот (эмпирически);
- Из теоретических соображений;
- Субъективно (экспертная оценка).

**Шаг 4**: Проверить аксиомы

- Неотрицательность: $P(A) \geq 0$;
- Нормировка: $P(\Omega) = 1$;
- Аддитивность для несовместных событий.

### Адекватность модели

Модель считается **адекватной**, если:

1. **Предсказания согласуются с наблюдениями**: Частоты близки к вероятностям;
2. **Модель не противоречива**: Аксиомы выполнены;
3. **Модель полезна**: Позволяет делать выводы и принимать решения.

### Множественность моделей

Для одного физического процесса может быть **несколько моделей**:

- С разными пространствами Ω (зависит от того, что регистрируем);
- С разными вероятностями (разные предположения).

## Примеры

**Пример 1**: Подбрасывание монеты

**Эксперимент**: Один раз подбросить монету

**Модель**:

- $\Omega = {Г, Р}$;
- $\mathscr{A} = {\emptyset, {Г}, {Р}, \Omega}$ (все подмножества);
- $P({Г}) = 1/2$, $P({Р}) = 1/2$ (честная монета).

**Пример 2**: Нечестная монета

**Тот же эксперимент**, но другая модель:

- $\Omega = {Г, Р}$ (то же);
- $\mathscr{A}$ (та же);
- $P({Г}) = 0.7$, $P({Р}) = 0.3$ (несимметричная).

Разные модели для одного эксперимента!

**Пример 3**: Бросок кости

**Эксперимент**: Бросить кость один раз

**Модель**:

- $\Omega = {1, 2, 3, 4, 5, 6}$;
- $\mathscr{A}$ = все подмножества Ω ($2^6 = 64$ события);
- $P({i}) = 1/6$ для всех i (честная кость).

**Пример 4**: Разные пространства для одного процесса

**Физический процесс**: Трёхкратное подбрасывание монеты

**Модель 1** (регистрируем все результаты):

- $\Omega_1 = {ГГГ, ГГР, \ldots, РРР}$ (8 исходов)

**Модель 2** (регистрируем только число гербов):

- $\Omega_2 = {0, 1, 2, 3}$ (4 исхода)

**Модель 3** (регистрируем только чётность числа гербов):

- $\Omega_3 = {\text{чёт}, \text{нечёт}}$ (2 исхода)

Разные модели → разные вопросы и вычисления!

**Пример 5**: Схема Бернулли

**Эксперимент**: n независимых испытаний, вероятность успеха p

**Модель**:

- $\Omega = {(a_1, \ldots, a_n) : a_i \in {0, 1}}$ (упорядоченные наборы);
- $|\Omega| = 2^n$;
- $p(\omega) = p^k q^{n-k}$, где k — число успехов в $\omega$.

**Пример 6**: Урна с шарами

**Эксперимент**: Из урны с 10 белыми и 5 чёрными извлекают 3 шара без возвращения

**Модель**:

- $\Omega$ = неупорядоченные тройки шаров
- $|\Omega| = C_{15}^3 = 455$
- $p(\omega) = 1/455$ (все тройки равновероятны при случайном выборе)
