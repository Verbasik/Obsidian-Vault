## Суть

**Сумма несовместных событий** A + B — это объединение событий A и B, когда они не могут произойти одновременно ($A \cap B = \emptyset$).

Обозначение A + B подчёркивает, что события **несовместны**, и их вероятности складываются: $$ P(A + B) = P(A) + P(B) $$

---

## Детали

### Определение

События A и B называются **несовместными**, если $A \cap B = \emptyset$.

Для несовместных событий объединение обозначается **суммой**: $$ A + B = A \cup B, \quad \text{если } A \cap B = \emptyset $$

### Отличие от обычного объединения

**Объединение** $A \cup B$ — для любых событий

**Сумма** $A + B$ — только для несовместных событий ($A \cap B = \emptyset$)

**Обозначение $A + B$** подчёркивает несовместность и указывает на аддитивность вероятности.

### Аддитивность вероятности

Для несовместных событий: $$ P(A + B) = P(A) + P(B) $$

Это одна из **аксиом Колмогорова** (аксиома аддитивности).

### Обобщение на n событий

Для попарно несовместных событий $A_1, \ldots, A_n$: $$ A_1 + A_2 + \cdots + A_n = \bigcup_{i=1}^n A_i $$

и $$ P(A_1 + \cdots + A_n) = P(A_1) + \cdots + P(A_n) $$

### Полная группа событий

События $A_1, \ldots, A_n$ образуют **полную группу (систему)**, если:

1. Они попарно несовместны: $A_i \cap A_j = \emptyset$ при $i \neq j$;
2. Их сумма — достоверное событие: $A_1 + \cdots + A_n = \Omega$.

Для полной группы: $$ P(A_1) + \cdots + P(A_n) = 1 $$

---

## Примеры

**Пример 1**: Бросок кости

$A = {1, 2}$ (число ≤ 2)

$B = {5, 6}$ (число ≥ 5)

$A \cap B = \emptyset$ — несовместны ✓

$A + B = {1, 2, 5, 6}$

$$ P(A + B) = P(A) + P(B) = \frac{2}{6} + \frac{2}{6} = \frac{4}{6} = \frac{2}{3} $$

**Пример 2**: Извлечение карты

$A$ = «туз» (4 карты)

$B$ = «король» (4 карты)

$A \cap B = \emptyset$ (карта не может быть одновременно тузом и королём)

$A + B$ = «туз или король»

$$ P(A + B) = \frac{4}{52} + \frac{4}{52} = \frac{8}{52} = \frac{2}{13} $$

**Пример 3**: Полная группа (разбиение)

Бросок кости:

$A_1 = {1, 2}$, $A_2 = {3, 4}$, $A_3 = {5, 6}$

Попарно несовместны: $A_i \cap A_j = \emptyset$ при $i \neq j$ ✓

Сумма = $\Omega$: $A_1 + A_2 + A_3 = {1,2,3,4,5,6} = \Omega$ ✓

Полная группа!

Проверка: $$ P(A_1) + P(A_2) + P(A_3) = \frac{2}{6} + \frac{2}{6} + \frac{2}{6} = 1 $$

**Пример 4**: Событие и его дополнение

$A$ — любое событие

$\bar{A}$ — дополнение

$A \cap \bar{A} = \emptyset$ — несовместны ✓

$A + \bar{A} = \Omega$ — образуют полную группу ✓

$$ P(A) + P(\bar{A}) = 1 $$

**Пример 5**: Неправильное использование

$A = {1, 2, 3}$, $B = {2, 4, 6}$

$A \cap B = {2} \neq \emptyset$ — **НЕ несовместны!**

Нельзя писать $A + B$ ✗

Правильно: $A \cup B$

$$ P(A \cup B) = P(A) + P(B) - P(A \cap B) = \frac{3}{6} + \frac{3}{6} - \frac{1}{6} = \frac{5}{6} $$

(не просто сумма!)


**Пример 6**: Подбрасывание монеты

$A$ = «герб» = ${Г}$

$B$ = «решка» = ${Р}$

$A \cap B = \emptyset$ ✓

$A + B = \Omega$ (полная группа)

$$ P(A) + P(B) = \frac{1}{2} + \frac{1}{2} = 1 $$

**Пример 7**: Схема Бернулли

n испытаний, считаем число успехов.

$A_k$ = «ровно k успехов», $k = 0, 1, \ldots, n$

События $A_0, A_1, \ldots, A_n$ попарно несовместны (не может быть одновременно k и m успехов при $k \neq m$)

$$ A_0 + A_1 + \cdots + A_n = \Omega $$

Полная группа!

$$ \sum_{k=0}^n P(A_k) = \sum_{k=0}^n C_n^k p^k q^{n-k} = (p + q)^n = 1^n = 1 $$
