## Суть

**Равновозможные исходы** — это элементарные события, которые имеют одинаковую вероятность в силу симметрии или однородности эксперимента.

Для конечного пространства $\Omega = {\omega_1, \ldots, \omega_N}$ с равновозможными исходами: $$ p(\omega_i) = \frac{1}{N} \quad \text{для всех } i = 1, \ldots, N $$

Это основа **классического определения вероятности**.

---

## Детали

### Определение

Элементарные события $\omega_1, \ldots, \omega_N$ называются **равновозможными** (или **равновероятными**), если: $$ p(\omega_1) = p(\omega_2) = \cdots = p(\omega_N) = \frac{1}{N} $$

### Условия равновозможности

Равновозможность предполагается, когда:

1. **Симметрия**: Нет физических причин предпочесть один исход другому;
2. **Однородность**: Условия эксперимента одинаковы для всех исходов;
3. **Отсутствие информации**: Нет оснований считать одни исходы более вероятными.

### Классическое определение вероятности

Для равновозможных исходов вероятность события A: $$ P(A) = \frac{|A|}{|\Omega|} = \frac{\text{число благоприятных исходов}}{\text{общее число исходов}} $$

где $|A|$ — число элементарных событий в A.

### Проверка нормировки

Сумма вероятностей всех элементарных событий: $$ \sum_{i=1}^N p(\omega_i) = N \cdot \frac{1}{N} = 1 $$

### Связь с комбинаторикой

При равновозможных исходах вычисление вероятностей сводится к **комбинаторному подсчёту**:

- Подсчитать общее число исходов $N = |\Omega|$;
- Подсчитать число благоприятных исходов $|A|$;
- Вычислить отношение $P(A) = |A|/N$.

### Примеры экспериментов с равновозможными исходами

1. **Симметричная монета**: $P(Г) = P(Р) = 1/2$;
2. **Честная кость**: $P({i}) = 1/6$ для $i = 1, \ldots, 6$;
3. **Случайный выбор**: извлечение наугад одного элемента из множества;
4. **Лотерея** (при случайном розыгрыше): все билеты равновероятны.

### Важные замечания

**Равновозможность — это предположение**, основанное на:

- Физической симметрии (правильная монета, кость);
- Случайности процесса (тщательное перемешивание);
- Отсутствии дополнительной информации.

**Реальность может отличаться**:

- Монета может быть неправильной (центр тяжести смещён);
- Кость может быть асимметричной;
- Колода может быть недостаточно перемешана.

## Примеры

**Пример 1**: Подбрасывание симметричной монеты

$\Omega = {Г, Р}$, $N = 2$

Равновозможные исходы: $p(Г) = p(Р) = 1/2$

Событие $A = {Г}$: $$ P(A) = \frac{|A|}{|\Omega|} = \frac{1}{2} $$

**Пример 2**: Бросок честной кости

$\Omega = {1, 2, 3, 4, 5, 6}$, $N = 6$

Равновозможные исходы: $p(i) = 1/6$ для всех i

Событие $A = {2, 4, 6}$ (четное число): $$ P(A) = \frac{|A|}{|\Omega|} = \frac{3}{6} = \frac{1}{2} $$

**Пример 3**: Двукратное подбрасывание монеты

$\Omega = {ГГ, ГР, РГ, РР}$, $N = 4$

Равновозможные исходы: $p(\omega) = 1/4$ для всех $\omega$

Событие $A$ = «хотя бы один герб» = ${ГГ, ГР, РГ}$: $$ P(A) = \frac{|A|}{|\Omega|} = \frac{3}{4} $$

**Пример 4**: Извлечение карты из колоды

$\Omega$ = все 52 карты, $N = 52$

При случайном извлечении все карты равновероятны: $p(\omega) = 1/52$

Событие $A$ = «извлечён туз»: $$ P(A) = \frac{4}{52} = \frac{1}{13} $$

**Пример 5**: Лотерея

1000 билетов, 10 выигрышных.

При случайном розыгрыше все билеты равновероятны: $p(\omega) = 1/1000$

Вероятность выигрыша: $$ P(\text{выигрыш}) = \frac{10}{1000} = \frac{1}{100} = 0.01 $$

**Пример 6**: Когда равновозможность НЕ выполнена

**Задача**: Семья с двумя детьми. Какова вероятность, что оба мальчики?

**Неправильное решение** (предполагая равновозможность): «Три возможности: ММ, МД, ДД. Вероятность ММ равна 1/3».

**Ошибка**: Исходы МД и ДМ различны! (старший/младший)

**Правильное решение**: $\Omega = {ММ, МД, ДМ, ДД}$ — 4 **равновозможных** исхода.

$P(ММ) = 1/4$


**Пример 7**: Сумма двух костей

При бросании двух костей сумма может быть от 2 до 12.

**Неправильно**: «11 возможных сумм, значит $P(\text{сумма} = 7) = 1/11$»

**Ошибка**: Суммы **НЕ равновозможны**!

**Правильно**: $\Omega$ — пары $(i, j)$, $1 \leq i, j \leq 6$. Всего $6 \times 6 = 36$ **равновозможных** исходов.

Сумма 7: $(1,6), (2,5), (3,4), (4,3), (5,2), (6,1)$ — 6 исходов.

$$ P(\text{сумма} = 7) = \frac{6}{36} = \frac{1}{6} $$

    > «Существует много практических ситуаций, в которых из соображений симметрии или однородности представляется разумным рассматривать все мыслимые исходы как равновозможные. Поэтому, если пространство элементарных исходов Ω состоит из точек $\omega_1, \ldots, \omega_N$, где $N < \infty$, то полагают $p(\omega_1) = ... = p(\omega_N) = 1/N$, и, следовательно, для любого события $A \in \mathscr{A}$: $P(A) = N(A)/N$, где $N(A)$ — число элементарных событий, составляющих A».
