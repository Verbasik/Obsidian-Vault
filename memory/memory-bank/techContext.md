# Tech Context

## Среда
- Хранилище: Obsidian Vault (Markdown заметки).
- Текущие артефакты кода: не обнаружены в корне и подкаталогах (сканирование по типовым индикаторам репозиториев и файлам кода не выявило результатов).

## Стек (на старте)
- Документация/Заметки: Markdown (Obsidian).
- Повторение: Anki (CSV экспорт в папках `Anki/anki_cards.csv`).
- Организация знаний: Zettelkasten (атомарные заметки, MOC, связи/ссылки).
- Инструменты: Obsidian, Anki; для автоматизации возможны локальные скрипты (будут добавлены при необходимости).
- Контроль версий: не зафиксирован (опционально Git поверх хранилища заметок).

## Интеграции и внешние зависимости
- Пока не используются. Возможные направления: AnkiConnect (локально), парсинг Markdown → CSV, граф связей.

## Политики качества
- Приоритеты P0/P1/P2 согласно протоколу.
- STaR для задач P0–P1; CoT фиксируется в артефактах, когда уместно.
- Шаблон заметки: заголовок, контекст/источник, определение/идея, примеры, связи, потенциальные карточки.
