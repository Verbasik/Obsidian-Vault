# Project Brief

## Контекст
- Рабочая папка: Obsidian Vault (Markdown‑заметки, Zettelkasten, Anki CSV, MOC‑заметки, PDF‑материалы).
- Область: подготовка к PhD (STEM), предметные области — Linear Algebra, Mathematical Analysis, Probability, Core; есть раздел Y‑HandBook и заметки по статьям arXiv.
- Цель инициализации: создать Банк Памяти для управления учебным знанием и процессом подготовки (не кодовый репозиторий).
- На момент инициализации README в корне не обнаружен, кодовые артефакты отсутствуют.

## Цели проекта
- Обеспечить единый источник истины по учебным целям, прогрессу и решениям.
- Организовать конвейер «Zettelkasten → Anki» для систематического закрепления знаний.
- Сформировать дорожную карту подготовки (темы, покрытия, дедлайны) и измеримые метрики мастерства.
- Поддержка принятия решений и планирования по P0–P2 приоритетам.

## Основные требования
- Структура Банка Памяти по протоколу: projectbrief, productContext, techContext, systemPatterns, activeContext, progress, rules/memory-bank.mdc.
- Обновление по триггерам (фичи, архитектурные изменения, новые требования, статус, паттерны, явный запрос).
- Методология STaR и фиксация знаний в журнале.

## Границы и допущения
- Текущая папка не содержит активного исходного кода; при появлении кода/репозитория контекст будет расширен.
- Банк Памяти хранится в `memory/` внутри текущего хранилища.

## Заинтересованные стороны
- Пользователь (владелец хранилища/проекта).
- Коллектив ИИ‑агентов (используют Банк Памяти как источник истины).

## KPI / Метрики
- Актуальность записей (P0 файлы согласованы и непротиворечивы).
- Полнота обновлений по триггерам.
- Время восстановления контекста (<2 минуты).
- Учебные метрики: покрытие тем по направлениям (%), темп появления атомарных заметок/неделю, доля связанных заметок (%), Anki‑retention (7/30/90 дней), streak ежедневных повторений.

## Риски и ограничения
- Недостаток исходного контекста (нет README/кода) → требуется уточнение от пользователя.
- Возможные противоречия в будущих файлах → разрешаются по правилу приоритетов (activeContext > progress > прочее).

## Источники правды
- Банк Памяти: `memory/memory-bank/` и `memory/rules/memory-bank.mdc`.
