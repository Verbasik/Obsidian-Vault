# Product Context

## Описание продукта
- Обучающее хранилище Obsidian для подготовки к PhD (STEM), построенное на методологии Zettelkasten с интеграцией Anki для интервальных повторений.
- Структура папок отражает предметные области: Linear Algebra, Mathematical Analysis, Probability, Core; отдельный раздел Y‑HandBook; материалы arXiv.

## Проблемы
- Потеря учебного контекста между сессиями и рассинхрон между заметками и карточками Anki.
- Неоднородность шаблонов заметок и отсутствующие связи между концептами.
- Недостаточная прозрачность покрытия тем и прогресса подготовки.

## Решения
- Структурированный Банк Памяти для управления учебным циклом (цели, прогресс, паттерны).
- Конвейер «Zettelkasten → Anki CSV»: атомарные заметки → извлечение карточек → регулярные повторения.
- Карта покрытия тем (coverage map) по каждому направлению с метриками.

## Пользовательские сценарии
- Добавить новую атомарную заметку с шаблона по теме и связать её с MOC/соседними концептами.
- Сгенерировать/обновить карточки в `anki_cards.csv` для раздела, провести дневной повтор.
- Просмотреть карту покрытия и решить, какие темы учить дальше.
- Конспектировать статью arXiv в формате Zettelkasten и связать с существующими узлами знаний.

## Метрики успеха
- Количество новых атомарных заметок/неделю; доля связанных заметок (%).
- Покрытие тем по направлениям (%), глубина (уровни связей в графе).
- Anki retention (7/30/90 дней), ежедневный streak повторений.

## Ограничения/неопределенности
- Нет кода/скриптов в хранилище: автоматизация планируется, но пока ручной процесс.
- Детали экзаменационных требований/дедлайнов — требуют уточнения.
